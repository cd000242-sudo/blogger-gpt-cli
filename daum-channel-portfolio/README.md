# 리더남 콘텐츠 운영 포트폴리오 제출 준비

이 폴더는 다음채널 개설 신청서의 `활동 중인 콘텐츠 자료`에 넣을 대표 포트폴리오 페이지용 파일입니다.

## 파일 구성

- `index.html`: 브라우저에서 바로 열 수 있는 완성형 페이지입니다.
- `wordpress-html.txt`: 워드프레스 고정 페이지의 사용자 정의 HTML 블록에 붙여넣는 버전입니다.
- `assets/`: 실제 운영 캡처 이미지가 들어 있는 폴더입니다.

## 포함된 이미지

- `assets/01-youtube-channel.png`: 유튜브 채널 운영 화면
- `assets/02-youtube-blog-tips.png`: 블로그 운영 노하우 영상 화면
- `assets/03-naver-blog-main.png`: 네이버 블로그 메인 화면
- `assets/04-naver-post-sample.png`: 네이버 정책 글 예시
- `assets/05-wordpress-article.png`: 워드프레스 글 예시
- `assets/06-blogspot-article.png`: 블로그스팟 본문 구성 예시
- `assets/06-1-blogspot-list.png`: 블로그스팟 글 목록 예시
- `assets/07-tool-screen.png`: 콘텐츠 운영 도구 화면

페이지에서는 이미지를 `background-image` 방식으로 표시합니다. 이미지가 빠져도 깨진 이미지 아이콘이 보이지 않고 회색 캡처 영역으로 남습니다.

## 이미지 교체 방법

새 캡처로 바꿀 때는 같은 파일명으로 `assets` 폴더에 덮어쓰면 됩니다.

워드프레스에 올릴 때는 이미지를 미디어 라이브러리에 업로드한 뒤, `wordpress-html.txt` 안의 `background-image` 주소를 업로드된 이미지 주소로 바꾸세요.

이미지에서 보여주고 싶은 영역이 어긋나면 해당 캡처의 `--pos: center 30%;` 값을 조정하면 됩니다. 두 번째 값이 작을수록 위쪽, 클수록 아래쪽이 보입니다.

## 다음채널 신청서에 URL 넣는 순서

1. `index.html`을 웹에 올리거나 워드프레스 고정 페이지로 발행합니다.
2. 대표 콘텐츠와 운영 채널 링크가 실제 주소로 바뀌었는지 확인합니다.
3. 모바일 화면에서 제목, 카드, 이미지 영역이 깨지지 않는지 확인합니다.
4. 발행된 페이지 URL을 복사합니다.
5. 다음채널 신청서의 `활동 중인 콘텐츠 자료` 또는 포트폴리오 URL 입력란에 붙여넣습니다.

## 제출 전 점검 체크리스트

- 페이지가 판매용 랜딩페이지처럼 보이지 않는지 확인
- 구매, 상담, 가격, 수익 보장처럼 심사에 불리할 수 있는 표현이 없는지 확인
- `AI`, `자동화`, `대량발행`, `수익화`, `상위노출 보장` 같은 단어가 본문에 남아 있지 않은지 확인
- 대표 콘텐츠 링크가 모두 실제 URL로 교체됐는지 확인
- 하단 대표 운영 채널 링크가 실제 URL로 교체됐는지 확인
- 지원금·정책 글은 공식 자료 확인 원칙이 보이는지 확인
- 인물 콘텐츠는 루머와 사생활 추측을 배제한다는 원칙이 보이는지 확인
- 이미지가 없을 때도 회색 캡처 영역이 자연스럽게 보이는지 확인
- 모바일에서 카드가 한 줄씩 정리되고 텍스트가 잘리지 않는지 확인
- 다음채널 심사자가 30초 안에 운영 이력과 콘텐츠 방향을 이해할 수 있는지 확인
