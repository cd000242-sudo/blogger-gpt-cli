# 전체 테스트 실행 안내

## ⏱️ 예상 소요 시간

**실제 소요 시간: 약 30-60분**

### 시간 구성:
1. **AI 모델 호출** (각 섹션마다): 10-30초 × 5-7개 섹션 = **5-15분**
2. **CTA URL 검증** (404 체크): 3초 × 2-3개 URL = **6-9초**
3. **이미지 생성** (선택 시): 10-20초 × 1개 = **10-20초**
4. **HTML 생성 및 최적화**: **1-2분**
5. **상세 검증**: **1-2분**

**총 예상 시간: 30-60분** (네트워크 상태에 따라 다름)

## 🚀 실행 방법
