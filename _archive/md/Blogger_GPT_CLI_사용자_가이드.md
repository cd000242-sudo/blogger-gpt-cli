## 🎯 앱 소개 및 시작하기

### Blogger GPT CLI란?
AI를 활용한 자동 블로그 포스팅 도구로, 키워드만 입력하면 SEO 최적화된 제목과 본문을 자동 생성하고 WordPress 또는 블로그스팟에 직접 발행할 수 있습니다.

### 주요 특징
- 🤖 **AI 자동 콘텐츠 생성**: Gemini AI를 활용한 고품질 블로그 포스팅
- 🎨 **썸네일 자동 생성**: 텍스트 기반 썸네일 이미지 생성
- 📊 **엑셀 대량 포스팅**: 엑셀 파일을 통한 대량 포스팅 관리
- 📅 **스케줄 포스팅**: 예약 포스팅 및 자동화
- 🔗 **다중 플랫폼 지원**: WordPress, 블로그스팟 연동

## ⚙️ 초기 설정

### 1단계: 앱 실행
```
1. Blogger GPT CLI 실행
2. 라이선스 확인 (상단 우측)
3. 환경설정 탭으로 이동
```

### 2단계: API 키 설정
#### 필수 API 키:
1. **Gemini API** (필수)
   - [Google AI Studio](https://makersuite.google.com/app/apikey) 방문
   - "Create API Key" 클릭
   - 생성된 키를 "Gemini API" 필드에 입력

2. **Pexels API** (이미지용)
   - [Pexels API](https://www.pexels.com/api/) 방문
   - 무료 계정 생성 후 API 키 발급
   - "Pexels API" 필드에 입력

3. **Google CSE** (검색용)
   - [Google Custom Search](https://cse.google.com/) 방문
   - 검색 엔진 생성
   - API 키와 검색 엔진 ID 입력

### 3단계: 블로그 플랫폼 설정

#### WordPress 설정:
```
1. 플랫폼 선택: "WordPress" 선택
2. 사이트 URL: https://yoursite.com
3. 사용자명: WordPress 관리자 계정
4. Application Password: WordPress에서 생성한 앱 비밀번호
```

#### 블로그스팟 설정:
```
1. 플랫폼 선택: "블로그스팟" 선택
2. Blogger ID: 블로그 ID (URL에서 확인)
3. Google Client ID: Google Cloud Console에서 생성
4. Google Client Secret: Google Cloud Console에서 생성
5. "블로거 계정 연동" 버튼 클릭
```

## 🏠 메인 기능 사용법

### 기본 포스팅 워크플로우

#### 1. 키워드 입력
```
메인 탭 → 키워드 입력란에 검색하고 싶은 키워드 입력
예: "맛집 추천", "여행 가이드", "IT 뉴스"
```

#### 2. 제목 생성
```
1. "제목 생성" 버튼 클릭
2. AI가 SEO 최적화된 제목 자동 생성
3. 여러 제목 옵션 중 선택 또는 수정
```

#### 3. 본문 생성
```
1. "본문 생성" 버튼 클릭
2. AI가 키워드 기반으로 상세한 본문 생성
3. 소제목, 목록, 결론 등 구조화된 콘텐츠 생성
```

#### 4. 썸네일 생성
```
썸네일 생성기 탭 → 텍스트 썸네일 생성기
1. 제목과 부제목 입력
2. 배경색, 텍스트 색상 선택
3. "썸네일 생성" 버튼 클릭
```

#### 5. 포스팅 실행
```
포스팅 실행 탭
1. 플랫폼 연동 확인
2. "포스팅 시작" 버튼 클릭
3. 실시간 진행 상황 모니터링
4. 완료 후 결과 확인
```

## 🎨 고급 기능 활용

### 1. 썸네일 생성기

#### 텍스트 썸네일 생성:
```
1. 썸네일 생성기 탭 선택
2. "텍스트 썸네일 생성기" 클릭
3. 제목과 부제목 입력
4. 스타일 설정:
   - 배경색: 그라데이션 또는 단색
   - 텍스트 색상: 가독성 고려
   - 폰트: 굵기, 크기 조정
5. 미리보기 확인 후 생성
```

#### 이미지 변환:
```
1. "이미지 변환기" 클릭
2. 이미지 파일 업로드
3. 썸네일 크기 설정 (예: 1200x630)
4. 변환 옵션 선택
5. "변환 시작" 버튼 클릭
```

### 2. 엑셀 대량 포스팅

#### 엑셀 파일 준비:
```
필수 컬럼:
- 키워드: 검색 키워드
- 제목: 포스팅 제목
- 본문: 포스팅 내용

선택 컬럼:
- 카테고리: WordPress 카테고리
- 태그: 포스팅 태그
- 발행일시: 예약 발행 시간
```

#### 대량 포스팅 실행:
```
1. 엑셀 포스팅 탭 선택
2. "엑셀 파일 선택" 버튼 클릭
3. 준비된 엑셀 파일 업로드
4. 데이터 미리보기 및 검증
5. "대량 포스팅 시작" 버튼 클릭
6. 진행 상황 모니터링
```

### 3. 스케줄 관리

#### 스케줄 설정:
```
1. 스케줄 관리 탭 선택
2. "새 스케줄" 버튼 클릭
3. 포스팅 콘텐츠 선택
4. 발행 날짜 및 시간 설정
5. 반복 설정 (일회성/반복)
6. "스케줄 저장" 버튼 클릭
```

#### 일정 확인:
```
1. 캘린더 뷰에서 예약된 포스팅 확인
2. 월별/주별/일별 뷰 전환
3. 스케줄 수정 또는 삭제
4. 자동 포스팅 상태 모니터링
```

## 🔧 환경설정 상세 가이드

### API 키 관리

#### Gemini API 설정:
```
1. Google AI Studio (https://makersuite.google.com/app/apikey) 방문
2. Google 계정으로 로그인
3. "Create API Key" 클릭
4. 생성된 키를 앱에 입력
5. "연동 확인" 버튼으로 테스트
```

#### Pexels API 설정:
```
1. Pexels (https://www.pexels.com/api/) 방문
2. "Get Started" 클릭하여 계정 생성
3. API 키 복사
4. 앱의 "Pexels API" 필드에 입력
```

#### Google CSE 설정:
```
1. Google Custom Search (https://cse.google.com/) 방문
2. "Add" 버튼으로 새 검색 엔진 생성
3. 검색할 사이트 URL 입력
4. API 키는 Google Cloud Console에서 생성
5. 검색 엔진 ID와 API 키를 앱에 입력
```

### 플랫폼별 상세 설정

#### WordPress 설정:
```
1. WordPress 관리자 → 사용자 → 프로필
2. "Application Passwords" 섹션 찾기
3. 새 애플리케이션 비밀번호 생성
4. 생성된 비밀번호를 앱에 입력
5. "연동 확인" 버튼으로 테스트
```

#### 블로그스팟 설정:
```
1. Google Cloud Console (https://console.cloud.google.com/) 방문
2. 새 프로젝트 생성 또는 기존 프로젝트 선택
3. "API 및 서비스" → "사용 설정된 API" → "Google+ API" 활성화
4. "사용자 인증 정보" → "OAuth 2.0 클라이언트 ID" 생성
5. 애플리케이션 유형: "데스크톱 애플리케이션"
6. 생성된 Client ID와 Secret을 앱에 입력
7. "블로거 계정 연동" 버튼으로 OAuth 인증
```

## 🚨 문제 해결

### 자주 발생하는 문제

#### 1. API 키 오류
```
증상: "API 키가 유효하지 않습니다" 오류
해결방법:
- API 키가 올바르게 복사되었는지 확인
- API 키에 공백이나 특수문자가 포함되지 않았는지 확인
- API 서비스에서 키가 활성화되어 있는지 확인
```

#### 2. 플랫폼 연동 실패
```
WordPress 연동 실패:
- 사이트 URL이 정확한지 확인 (https:// 포함)
- Application Password가 올바른지 확인
- WordPress에서 REST API가 활성화되어 있는지 확인

블로그스팟 연동 실패:
- Google Cloud Console에서 OAuth 설정 확인
- 리디렉션 URI 설정 확인
- Google 계정 권한 확인
```

#### 3. 포스팅 실패
```
증상: 포스팅이 시작되지 않거나 중단됨
해결방법:
- 네트워크 연결 상태 확인
- 플랫폼 연동 상태 재확인
- 로그 메시지에서 구체적인 오류 확인
- 앱 재시작 후 재시도
```

#### 4. 썸네일 생성 실패
```
증상: 썸네일이 생성되지 않음
해결방법:
- 이미지 크기가 너무 크지 않은지 확인
- 지원되는 이미지 형식인지 확인 (JPG, PNG)
- Pexels API 키가 올바른지 확인
```

### 로그 확인 방법
```
1. 앱 하단의 로그 창에서 오류 메시지 확인
2. 빨간색 오류 메시지에 주의
3. 구체적인 오류 내용을 기록
4. 필요시 앱 재시작
```

## ❓ FAQ

### Q1: 무료로 사용할 수 있나요?
A: 네, 기본 기능은 무료로 사용 가능합니다. 하지만 고급 기능과 대량 포스팅을 위해서는 라이선스 구매가 필요합니다.

### Q2: 어떤 블로그 플랫폼을 지원하나요?
A: 현재 WordPress와 블로그스팟을 지원합니다. 향후 더 많은 플랫폼 지원 예정입니다.

### Q3: AI가 생성한 콘텐츠의 품질은 어떤가요?
A: Gemini AI를 사용하여 고품질의 SEO 최적화된 콘텐츠를 생성합니다. 하지만 필요에 따라 수동으로 편집하는 것을 권장합니다.

### Q4: 대량 포스팅 시 제한이 있나요?
A: 플랫폼별 API 제한이 있습니다. WordPress는 일반적으로 분당 60회, 블로그스팟은 분당 100회 정도의 제한이 있습니다.

### Q5: 오프라인에서 사용할 수 있나요?
A: AI 기능과 포스팅 기능은 인터넷 연결이 필요합니다. 하지만 설정과 데이터는 로컬에 저장됩니다.

### Q6: 백업은 어떻게 하나요?
A: 환경설정 탭에서 "백업 생성" 버튼을 클릭하면 설정과 데이터를 백업할 수 있습니다.

### Q7: 여러 블로그를 관리할 수 있나요?
A: 네, 설정을 변경하여 여러 블로그를 번갈아 관리할 수 있습니다.

### Q8: 스케줄 포스팅은 앱이 실행 중이어야 하나요?
A: 네, 스케줄 포스팅은 앱이 실행 중이어야 정상 작동합니다.
