# 블로그 지수 분석 기능 구현 플랜

## 📋 개요
키워드를 입력하면 네이버 검색 결과에 나오는 블로그들의 순위별로 블로그 지수를 분석하고 표시하는 기능

---

## 🎯 목표
- 키워드 검색 결과 상위 블로그들의 지수 분석
- 경쟁 블로그 분석을 통한 전략 수립 지원
- 블로그 순위와 지수의 상관관계 파악

---

## 📊 블로그 지수 계산 방법

### 1. 지수 구성 요소

#### A. 기본 지표 (가중치 40%)
- **방문자 수** (가중치: 15%)
  - 일일/월간 방문자 추정치
  - 네이버 블로그 통계 API 또는 크롤링
  
- **포스팅 수** (가중치: 10%)
  - 전체 포스팅 개수
  - 최근 1개월 포스팅 빈도
  
- **구독자 수** (가중치: 10%)
  - 블로그 구독자 수
  - 네이버 블로그 프로필에서 추출
  
- **평균 조회수** (가중치: 5%)
  - 최근 포스팅들의 평균 조회수
  - 상위 10개 포스팅 기준

#### B. 활동도 지표 (가중치 30%)
- **최근 포스팅 빈도** (가중치: 15%)
  - 최근 1개월 포스팅 수
  - 최근 3개월 포스팅 수
  
- **최근 업데이트** (가중치: 10%)
  - 마지막 포스팅 날짜
  - 활동 지속성 점수
  
- **댓글/공감 수** (가중치: 5%)
  - 최근 포스팅들의 평균 댓글 수
  - 평균 공감 수

#### C. 품질 지표 (가중치 20%)
- **콘텐츠 길이** (가중치: 8%)
  - 평균 포스팅 길이
  - 상세한 콘텐츠 여부
  
- **이미지/미디어 사용** (가중치: 7%)
  - 포스팅당 평균 이미지 수
  - 동영상 포함 여부
  
- **SEO 최적화** (가중치: 5%)
  - 제목 최적화 여부
  - 태그 사용 여부

#### D. 키워드 관련성 (가중치 10%)
- **키워드 노출 빈도** (가중치: 5%)
  - 검색된 키워드가 제목/본문에 나타나는 빈도
  
- **키워드 순위** (가중치: 5%)
  - 검색 결과에서의 순위
  - 상위 노출 여부

### 2. 지수 계산 공식

```
블로그 지수 = 
  (기본 지표 점수 × 0.4) +
  (활동도 지표 점수 × 0.3) +
  (품질 지표 점수 × 0.2) +
  (키워드 관련성 점수 × 0.1)

각 지표 점수 = (실제값 / 최대값) × 100
```

interface BlogIndexScore {
  basicScore: number;      // 기본 지표 점수 (0-100)

## ⚠️ 주의사항 및 제약사항

### 1. API 제한
- **네이버 검색 API**: 일일 호출 제한 (25,000건)
- **Rate Limit**: 초당 요청 수 제한
- **해결책**: 요청 간 지연 시간 추가, 캐싱 활용

### 2. 크롤링 제한
- **로봇 차단**: User-Agent 설정 필수
- **IP 차단**: 프록시 사용 고려
- **법적 문제**: robots.txt 준수, 공개 정보만 수집

### 3. 데이터 정확도
- **추정치 사용**: 일부 정보는 추정치로 계산
- **업데이트 지연**: 실시간 데이터가 아닐 수 있음
- **명시**: 사용자에게 추정치임을 명확히 표시

### 4. 성능
- **병렬 처리**: 여러 블로그 정보를 동시에 수집
- **타임아웃**: 각 요청에 타임아웃 설정
- **진행률 표시**: 사용자에게 진행 상황 표시
