## 정리 방안

1. **공통 유틸 함수로 통합**
   - `sanitizeString` → 최상단 공통 함수로 이동
   - `removeNavigation` → 공통 함수로 통합
   - `removePartDividers` → 공통 함수로 통합
   - `removeDuplicateChecklistLinks` → 공통 함수로 통합
   - `removeDuplicateH2` → 공통 함수로 통합
   - `removeDuplicateParagraphs` → 공통 함수로 통합

2. **중복 제거 로직 통합**
   - 섹션 처리 중 중복 제거 로직 제거
   - 최종 HTML 정리에서만 한 번에 처리

3. **함수 호출 순서 최적화**
   - 한 곳에서만 처리하여 성능 향상 및 버그 방지
