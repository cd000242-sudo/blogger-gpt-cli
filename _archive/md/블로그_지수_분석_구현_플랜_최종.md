## 🔍 네이버 블로그 지수 조회 방법 조사

### 현재 상황
1. **네이버 블로그 검색 API**는 이미 사용 중
   - 엔드포인트: `https://openapi.naver.com/v1/search/blog.json`
   - 응답 필드: `title`, `link`, `description`, `bloggername`, `bloggerlink`, `postdate`

2. **네이버 공식 블로그 지수 API는 없음**
   - 웹 검색 결과: 네이버에서 공식적으로 제공하는 "블로그 지수" API는 존재하지 않음
   - 블로그 지수는 네이버 내부 지표로 공개되지 않음

### 가능한 해결 방법

#### 방법 1: 블로그 프로필 페이지 크롤링 (가장 현실적)
**URL 패턴**: `https://blog.naver.com/{blogId}`

**크롤링 대상**:
- 블로그 프로필 페이지에서 지수 정보 추출
- 네이버 블로그는 프로필 페이지에 "블로그 지수" 또는 "인기도" 표시
- HTML 파싱 또는 JavaScript 실행 필요

**구현 난이도**: ⭐⭐⭐ (중상)
**예상 시간**: 4-6시간

#### 방법 2: 블로그 검색 API 응답 확인 (우선 확인)
**확인 사항**:
- 네이버 블로그 검색 API 응답에 지수 관련 필드가 있는지 확인
- `item` 객체에 숨겨진 필드가 있는지 확인

**구현 난이도**: ⭐ (하)
**예상 시간**: 1-2시간 (확인만)

#### 방법 3: 대체 지표 사용 (대안)
**사용 가능한 지표**:
- 검색 순위 (이미 있음)
- 포스팅 수 (RSS 피드에서 추출)
- 최근 활동도 (포스팅 날짜 기반)
- 구독자 수 (프로필에서 추출 가능한 경우)

**구현 난이도**: ⭐⭐ (중)
**예상 시간**: 2-3시간

function extractBlogId(bloggerlink: string): string {
  // 예: "https://blog.naver.com/myblog" → "myblog"
  const match = bloggerlink.match(/blog\.naver\.com\/([^\/\?]+)/);
  return match ? match[1] : '';
}

#### 시나리오 B: 대체 지표 사용

```typescript
interface BlogIndexInfo {
  blogId: string;
  blogUrl: string;
  estimatedIndex: number;  // 추정 지수
  metrics: {
    searchRank: number;    // 검색 순위
    postCount?: number;    // 포스팅 수
    recentActivity?: number; // 최근 활동도
  };
}

function calculateEstimatedIndex(metrics: BlogIndexInfo['metrics']): number {
  // 검색 순위 기반 추정 지수
  // 순위가 높을수록 (숫자가 작을수록) 지수 높음
  const rankScore = Math.max(0, 100 - (metrics.searchRank * 2));
  
  // 포스팅 수 기반 점수 (있는 경우)
  const postScore = metrics.postCount 
    ? Math.min(50, (metrics.postCount / 100) * 50) 
    : 0;
  
  // 활동도 점수 (있는 경우)
  const activityScore = metrics.recentActivity || 0;
  
  return Math.round(rankScore + postScore + activityScore);
}
```

async function analyzeBlogIndex() {
  const keyword = document.getElementById('blogIndexKeyword').value.trim();
  if (!keyword) {
    alert('키워드를 입력해주세요');
    return;
  }
  
  const resultsDiv = document.getElementById('blogIndexResults');
  const loadingDiv = document.getElementById('blogIndexLoading');
  const tbody = document.getElementById('blogIndexTableBody');
  
  // UI 초기화
  resultsDiv.style.display = 'none';
  loadingDiv.style.display = 'block';
  tbody.innerHTML = '';
  
  try {
    const result = await window.electronAPI.invoke('analyze-blog-index', {
      keyword,
      maxResults: 50
    });
    
    if (result.ok && result.data) {
      renderBlogIndexTable(result.data);
      document.getElementById('blogIndexTotal').textContent = result.data.totalResults;
      resultsDiv.style.display = 'block';
    } else {
      alert('분석 실패: ' + (result.error || '알 수 없는 오류'));
    }
  } catch (error) {
    console.error('블로그 지수 분석 오류:', error);
    alert('오류 발생: ' + error.message);
  } finally {
    loadingDiv.style.display = 'none';
  }
}

function renderBlogIndexTable(data) {
  const tbody = document.getElementById('blogIndexTableBody');
  
  // 지수 순위 계산
  const blogsWithIndex = data.blogs
    .filter(blog => blog.blogIndex !== null && blog.blogIndex !== undefined)
    .sort((a, b) => b.blogIndex - a.blogIndex);
  
  blogsWithIndex.forEach((blog, idx) => {
    blog.indexRank = idx + 1;
  });
  
  // 검색 순위별로 정렬하여 표시
  const sortedBlogs = data.blogs.sort((a, b) => a.searchRank - b.searchRank);
  
  tbody.innerHTML = sortedBlogs.map(blog => `
    <tr>
      <td style="padding: 8px; border: 1px solid #ddd;">${blog.searchRank}</td>
      <td style="padding: 8px; border: 1px solid #ddd;">
        <a href="${blog.blogUrl}" target="_blank" style="color: #0066cc; text-decoration: none;">
          ${blog.blogName}
        </a>
      </td>
      <td style="padding: 8px; border: 1px solid #ddd; text-align: center;">
        ${blog.blogIndex !== null && blog.blogIndex !== undefined 
          ? `<strong style="color: #4CAF50;">${blog.blogIndex}</strong>` 
          : '<span style="color: #999;">N/A</span>'}
      </td>
      <td style="padding: 8px; border: 1px solid #ddd; text-align: center;">
        ${blog.indexRank || '-'}
      </td>
    </tr>
  `).join('');
}

### **Step 5: IPC 핸들러 추가** (30분)

**파일**: `electron/main.js`

```javascript
ipcMain.handle('analyze-blog-index', async (_event, { keyword, maxResults = 50 }) => {
  try {
    const { EnvironmentManager } = await import('../src/utils/environment-manager');
    const envManager = EnvironmentManager.getInstance();
    const env = envManager.getConfig();
    
    if (!env.naverClientId || !env.naverClientSecret) {
      return { ok: false, error: '네이버 API 키가 설정되지 않았습니다.' };
    }
    
    const analyzer = await import('../src/utils/blog-index-analyzer');
    const result = await analyzer.analyzeBlogIndex(
      keyword,
      maxResults,
      {
        clientId: env.naverClientId,
        clientSecret: env.naverClientSecret
      }
    );
    
    return { ok: true, data: result };
  } catch (error) {
    console.error('[BLOG-INDEX] 분석 실패:', error);
    return { ok: false, error: error.message || '알 수 없는 오류' };
  }
});
```
