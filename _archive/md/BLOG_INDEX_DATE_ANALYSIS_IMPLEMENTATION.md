## 🎯 사용 방법

### 키워드 검색 시 자동 분석
1. 키워드 입력 후 검색
2. 상위 노출 글 목록 자동 표시 (최대 100개)
3. 작성일자 기반 기회 점수 자동 계산
4. 블로그 지수 자동 조회 (빠른 모드)

### 표시 정보
- **순위**: 네이버 검색 결과 순위 (1위부터)
- **상위노출 글 목록**: 실제 네이버 검색 결과 URL (100% 일치)
- **작성일**: 네이버 API에서 제공하는 정확한 날짜
- **기회 점수**: 작성일자 기반 분석 점수 (0-100점)
  - 🟢 80점 이상: 매우 좋은 기회
  - 🟡 60-79점: 좋은 기회
  - 🔴 40-59점: 보통 기회
  - ⚪ 40점 미만: 낮은 기회
- **블로그 지수**: 추정값 (88-99% 신뢰도)
- **신뢰도**: 블로그 지수 추정값의 신뢰도 표시
