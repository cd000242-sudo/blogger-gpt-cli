# 블로그 지수 분석 기능 - 상세 구현 플랜

## 📋 프로젝트 개요

**목표**: 키워드 검색 결과에 나오는 블로그들의 순위별 지수 분석 기능 구현

**예상 총 개발 기간**: **4-6주** (전일 근무 기준)
- 실제로는 부분 시간 작업이므로 **6-8주** 소요 가능
- 병렬 작업 가능한 부분을 최대한 활용하면 **4-5주**로 단축 가능

---

## 🎯 핵심 기능 정의

### MVP (Minimum Viable Product) - 1차 목표
1. 키워드 입력 → 네이버 블로그 검색 결과 수집 (상위 20-30개)
2. 각 블로그의 기본 정보 수집 (제목, 링크, 설명, 날짜)
3. 간단한 지수 계산 (포스팅 수, 최근 활동도, 키워드 관련성)
4. 결과 테이블 표시 (순위, 블로그명, 지수, 간단한 통계)

### Full Version - 2차 목표
1. 상세 정보 크롤링 (구독자, 조회수, 댓글 수 등)
2. 정교한 지수 계산 (4가지 지표 종합)
3. 시각화 차트
4. 상세 분석 리포트
5. 데이터 내보내기

**코드 구조**:
```typescript
// src/utils/blog-index-calculator.ts
interface BlogIndexScore {
  totalScore: number;      // 0-100
  postCountScore: number;   // 포스팅 수 점수
  activityScore: number;    // 활동도 점수
  relevanceScore: number;   // 키워드 관련성 점수
}

function calculateBasicBlogIndex(
  blogInfo: BlogBasicInfo,
  keyword: string,
  searchRank: number
): BlogIndexScore
```

**계산 공식 (MVP)**:
```
포스팅 수 점수 = min(100, (포스팅 수 / 100) * 100)
활동도 점수 = 최근 1개월 포스팅 수 * 10 (최대 50점)
관련성 점수 = (30 - 순위) * 3.33 (최대 100점)
종합 점수 = (포스팅 점수 * 0.4) + (활동도 * 0.3) + (관련성 * 0.3)
```

#### Day 4-5: 정교한 지수 계산
**작업 내용**:
- [ ] 4가지 지표 종합 계산
- [ ] 가중치 적용
- [ ] 정규화 및 점수 조정
- [ ] 상대적 점수 계산

**예상 시간**: 4-6시간
**난이도**: ⭐⭐⭐ (중상)

**계산 공식 (Full Version)**:
```
기본 지표 (40%):
  - 방문자 수 점수: (방문자 수 / 최대값) * 100 * 0.15
  - 포스팅 수 점수: (포스팅 수 / 최대값) * 100 * 0.10
  - 구독자 수 점수: (구독자 수 / 최대값) * 100 * 0.10
  - 평균 조회수 점수: (조회수 / 최대값) * 100 * 0.05

활동도 지표 (30%):
  - 최근 포스팅 빈도: (1개월 포스팅 수 / 최대값) * 100 * 0.15
  - 최근 업데이트: (마지막 포스팅 경과일 역수) * 100 * 0.10
  - 댓글/공감 수: (평균 댓글 수 / 최대값) * 100 * 0.05

품질 지표 (20%):
  - 콘텐츠 길이: (평균 글자 수 / 최대값) * 100 * 0.08
  - 이미지 사용: (평균 이미지 수 / 최대값) * 100 * 0.07
  - SEO 최적화: (제목/태그 최적화 여부) * 100 * 0.05

키워드 관련성 (10%):
  - 키워드 노출: (제목+본문 키워드 빈도) * 100 * 0.05
  - 검색 순위: (30 - 순위) * 3.33 * 0.05

종합 지수 = 모든 지표 점수 합계
```
