## ✅ **수정 내역**

### **1. `env:load` 핸들러 추가**
```typescript
// electron/main.ts에 추가
ipcMain.handle('env:load', async () => {
  try {
    const env = loadEnvFromFile();
    return { ok: true, env };
  } catch (error) {
    console.error('[ENV-LOAD] 로드 실패:', error);
    return { ok: false, error: error instanceof Error ? error.message : '로드 실패', env: {} };
  }
});
```

**효과**:
- keyword-master.html이 환경 변수를 정상적으로 불러올 수 있음
- API 키 상태 확인 가능
- 네이버/유튜브/검색광고 API 연동 상태 표시

### **2. Keyword Master 창 Preload 수정**
```typescript
// open-keyword-master-window 핸들러 개선
const keywordWindow = new BrowserWindow({
  width: 1400,
  height: 900,
  webPreferences: {
    nodeIntegration: false,
    contextIsolation: true,
    preload: path.join(__dirname, 'ui', 'preload.js'), // 명확한 경로
    webSecurity: false // CORS 문제 해결
  }
});

// 개발자 도구 자동 열기 (디버깅용)
keywordWindow.webContents.openDevTools();
```

**효과**:
- `window.electronAPI` 정상 로드
- `window.blogger` API 접근 가능

## 💡 **주의사항**

### **API 키 설정**
LEWORD의 모든 기능을 사용하려면 다음 API 키가 필요합니다:

1. **네이버 검색 API**
   - `.env` 파일에 `NAVER_CLIENT_ID`, `NAVER_CLIENT_SECRET` 추가

2. **유튜브 API**
   - `.env` 파일에 `YOUTUBE_API_KEY` 추가

3. **네이버 검색광고 API**
   - `.env` 파일에 `NAVER_AD_API_KEY`, `NAVER_AD_SECRET`, `NAVER_AD_CUSTOMER_ID` 추가

API 키가 없어도 실시간 검색어 등 일부 기능은 사용 가능합니다!
