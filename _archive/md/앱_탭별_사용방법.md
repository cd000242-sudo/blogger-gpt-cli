# 📱 Blogger GPT CLI - 탭별 사용방법 가이드

## 🏠 메인 탭
**기능**: 블로그 포스팅의 핵심 작업 공간

### 주요 기능:
- **실시간 시계 & 달력**: 현재 시간과 날짜 확인
- **키워드 관리**: 블로그 포스팅을 위한 키워드 입력 및 관리
- **콘텐츠 생성**: AI를 활용한 제목 및 본문 자동 생성
- **미리보기**: 생성된 콘텐츠 미리보기 및 편집

### 사용방법:
1. **키워드 입력**: 검색하고 싶은 키워드를 입력
2. **제목 생성**: AI가 자동으로 SEO 최적화된 제목 생성
3. **본문 생성**: 키워드 기반으로 상세한 블로그 포스팅 본문 생성
4. **미리보기**: 생성된 콘텐츠를 확인하고 필요시 수정
5. **포스팅**: WordPress 또는 블로그스팟에 직접 발행

## 🎨 썸네일 생성기 탭
**기능**: 블로그 포스팅용 썸네일 이미지 생성

### 주요 기능:
- **텍스트 썸네일 생성기**: 텍스트 기반 썸네일 이미지 생성
- **이미지 변환기**: 기존 이미지를 썸네일 형식으로 변환

### 사용방법:
1. **텍스트 썸네일 생성**:
   - 제목과 부제목 입력
   - 배경색, 텍스트 색상 선택
   - 폰트 스타일 및 크기 조정
   - "썸네일 생성" 버튼 클릭

2. **이미지 변환**:
   - 이미지 파일 업로드
   - 썸네일 크기 설정 (가로×세로)
   - 변환 옵션 선택
   - "변환 시작" 버튼 클릭

## ⚙️ 포스팅 실행 탭
**기능**: 블로그 플랫폼 설정 및 포스팅 실행

### 주요 기능:
- **플랫폼 설정**: WordPress 또는 블로그스팟 선택
- **인증 설정**: 각 플랫폼별 인증 정보 입력
- **포스팅 실행**: 생성된 콘텐츠를 실제 블로그에 발행

### 사용방법:

#### WordPress 설정:
1. **플랫폼 선택**: "WordPress" 라디오 버튼 선택
2. **인증 정보 입력**:
   - WordPress 사이트 URL
   - 사용자명
   - Application Password
3. **연동 확인**: "연동 확인" 버튼으로 연결 테스트

#### 블로그스팟 설정:
1. **플랫폼 선택**: "블로그스팟" 라디오 버튼 선택
2. **인증 정보 입력**:
   - Blogger ID
   - Google Client ID
   - Google Client Secret
3. **계정 연동**: "블로거 계정 연동" 버튼 클릭
4. **OAuth 인증**: Google 계정으로 로그인 및 권한 승인

#### 포스팅 실행:
1. **콘텐츠 확인**: 생성된 제목과 본문 검토
2. **포스팅 시작**: "포스팅 시작" 버튼 클릭
3. **진행 상황 모니터링**: 실시간 진행률 확인
4. **완료 확인**: 포스팅 완료 후 결과 확인

## 📊 엑셀 포스팅 탭
**기능**: 엑셀 파일을 통한 대량 포스팅 관리

### 주요 기능:
- **엑셀 파일 업로드**: 포스팅 데이터가 포함된 엑셀 파일 업로드
- **데이터 미리보기**: 엑셀 데이터 확인 및 검증
- **대량 포스팅**: 여러 포스팅을 한 번에 실행

### 사용방법:
1. **엑셀 파일 준비**:
   - 키워드, 제목, 본문 등이 포함된 엑셀 파일 생성
   - 필수 컬럼: 키워드, 제목, 본문
   - 선택 컬럼: 카테고리, 태그, 발행일시

2. **파일 업로드**:
   - "엑셀 파일 선택" 버튼 클릭
   - 준비된 엑셀 파일 선택

3. **데이터 확인**:
   - 업로드된 데이터 미리보기
   - 필수 필드 누락 여부 확인
   - 데이터 수정 (필요시)

4. **대량 포스팅 실행**:
   - "대량 포스팅 시작" 버튼 클릭
   - 진행 상황 모니터링
   - 완료 후 결과 리포트 확인

## 📅 스케줄 관리 탭
**기능**: 포스팅 스케줄링 및 일정 관리

### 주요 기능:
- **스케줄 설정**: 포스팅 발행 시간 예약
- **일정 관리**: 캘린더를 통한 포스팅 일정 확인
- **자동 포스팅**: 설정된 시간에 자동으로 포스팅 실행

### 사용방법:
1. **스케줄 생성**:
   - 포스팅할 콘텐츠 선택
   - 발행 날짜 및 시간 설정
   - 반복 설정 (일회성/반복)

2. **일정 확인**:
   - 캘린더에서 예약된 포스팅 확인
   - 월별/주별/일별 뷰 전환
   - 스케줄 수정 또는 삭제

3. **자동 포스팅**:
   - 백그라운드에서 자동 실행
   - 실시간 진행 상황 모니터링
   - 완료 알림 및 결과 확인

## 🔧 환경설정 탭
**기능**: 앱 설정 및 API 키 관리

### 주요 기능:
- **API 키 설정**: AI 모델 및 서비스 API 키 관리
- **플랫폼 설정**: 블로그 플랫폼 연결 설정
- **고급 설정**: 추가 옵션 및 사용자 정의 설정

### 사용방법:

#### API 키 설정:
1. **Gemini API**: Google AI Studio에서 발급받은 API 키 입력
2. **Pexels API**: 무료 이미지 서비스 API 키 입력
3. **DALL-E API**: OpenAI 이미지 생성 API 키 입력
4. **네이버 API**: 네이버 검색 API 키 입력

#### 플랫폼 설정:
1. **WordPress**: 사이트 URL, 사용자명, Application Password
2. **블로그스팟**: Blogger ID, Google OAuth 설정

#### 고급 설정:
1. **최소 글자 수**: AI 생성 콘텐츠의 최소 글자 수 설정
2. **썸네일 타입**: 기본 썸네일 생성 방식 선택
3. **백업 관리**: 설정 및 데이터 백업/복원

---

## 💡 사용 팁

### 효율적인 사용법:
1. **메인 탭**에서 기본 콘텐츠 생성
2. **썸네일 생성기**로 시각적 요소 추가
3. **포스팅 실행**에서 실제 발행
4. **엑셀 포스팅**으로 대량 작업 처리
5. **스케줄 관리**로 자동화 설정

### 주의사항:
- API 키는 정확히 입력해야 정상 작동
- 플랫폼별 인증 정보는 보안에 주의
- 대량 포스팅 시 네트워크 상태 확인
- 스케줄 포스팅은 앱이 실행 중이어야 함

## 🆘 문제 해결

### 자주 발생하는 문제:
1. **API 키 오류**: 키가 올바른지 확인
2. **연동 실패**: 플랫폼 인증 정보 재확인
3. **포스팅 실패**: 네트워크 연결 상태 확인
4. **썸네일 생성 실패**: 이미지 크기 및 형식 확인

### 지원:
- 로그 메시지를 통한 오류 진단
- 설정 초기화 및 재설정
- 백업 복원을 통한 문제 해결
