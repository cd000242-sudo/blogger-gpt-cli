# 크롤링 기능 연동 확인 체크리스트

## 1. Google Search 크롤링 연동 확인
- [ ] `useGoogleSearch` 옵션이 콘텐츠 생성 payload에 포함되는지 확인
- [ ] Google CSE API 키가 설정되어 있는지 확인
- [ ] 크롤링 결과가 콘텐츠에 반영되는지 확인

## 2. 네이버 API 크롤링 확인
- [ ] 네이버 Customer ID가 설정되어 있는지 확인
- [ ] 네이버 Secret Key가 설정되어 있는지 확인
- [ ] 네이버 검색 API 호출이 payload에 포함되는지 확인

## 3. RSS 기능 확인
- [ ] RSS 피드 URL 처리 기능 존재 여부 확인
- [ ] RSS 피드 파싱 기능 존재 여부 확인
- [ ] RSS 피드 데이터가 콘텐츠에 반영되는지 확인

## 테스트 방법
1. 콘텐츠 생성 시 콘솔에서 payload 확인
2. 크롤링 옵션 활성화 후 콘텐츠 생성
3. 생성된 콘텐츠에 외부 정보가 포함되었는지 확인
