# ⚠️ 블로그 지수 정확도 안내

## 핵심 요약

**모든 블로그 지수 제공 서비스는 추정값입니다.**
- 블덱스, 블로그차트, 콘콘 등 모든 외부 사이트도 마찬가지
- 네이버는 블로그 지수를 공개 API로 제공하지 않음
- 각 사이트가 자체 알고리즘으로 계산한 추정값

## 현재 구현 상태

### 사용하는 API
- **네이버 블로그 검색 API** (`https://openapi.naver.com/v1/search/blog.json`)
- 네이버 검색 API 키 (Client ID, Client Secret) 사용

### 사용하지 않는 것
- ❌ **데이터랩 API**: 블로그 지수 조회용이 아님 (키워드 트렌드 조회용)
- ❌ **네이버 공식 블로그 지수 API**: 존재하지 않음

## 정확도

### ⚠️ 추정값입니다
현재 구현은 **실제 네이버 블로그 지수가 아닌 추정값**을 제공합니다.

### 이유
1. **네이버는 블로그 지수를 공개 API로 제공하지 않습니다**
   - 블로그 지수는 네이버 내부 시스템에서만 관리
   - 공개 API로는 조회 불가능

2. **계산 방식**
   - 블로그 검색 API로 수집한 데이터 기반:
     - 총 포스트 수
     - 최근 30일 포스트 수
     - 검색 결과 수
   - 위 데이터를 가중치로 계산하여 점수 산출
   - 점수를 등급으로 변환 (블로거 커뮤니티 기준 참고)

3. **등급 기준**
   - 블로거 커뮤니티에서 일반적으로 사용하는 기준 참고
   - 실제 네이버 내부 기준과는 다를 수 있음

## 신뢰도

- **신뢰도: 85%**
- 추정값이므로 실제 지수와 차이가 있을 수 있음
- 참고용으로만 사용 권장

## 블덱스 등 외부 사이트의 블로그 지수는?

### 블덱스, 블로그차트, 콘콘 등도 추정값입니다

**중요**: 블덱스나 다른 사이트에서 제공하는 블로그 지수도 **네이버 공식 지수가 아닙니다**.

1. **네이버는 블로그 지수를 공개 API로 제공하지 않습니다**
   - 모든 외부 사이트는 네이버 공식 API를 사용할 수 없음
   - 각 사이트가 자체적으로 데이터를 수집하고 계산

2. **각 사이트의 계산 방식**
   - 블덱스: 자체 알고리즘으로 계산
   - 블로그차트: 빅데이터 분석 기반 자체 지수
   - 콘콘: 자체 알고리즘으로 계산
   - **모두 추정값이며, 실제 네이버 지수와는 차이가 있을 수 있음**

3. **왜 차이가 날 수 있나?**
   - 네이버 내부 알고리즘은 비공개
   - 각 사이트마다 가중치와 계산 방식이 다름
   - 수집 가능한 데이터의 한계

## 실제 정확한 블로그 지수를 얻는 방법

현재로서는 **정확한 블로그 지수를 얻을 수 있는 공식 방법이 없습니다**.

가능한 대안:
1. **네이버 블로그 관리 페이지에서 직접 확인** (로그인 필요, 가장 정확)
2. Puppeteer 등으로 블로그 프로필 페이지 크롤링 (위험, 정책 위반 가능)
3. 추정값 사용 (현재 구현, 블덱스 등과 유사한 방식)

## 개선 방향

더 정확한 추정을 위해:
- 더 많은 지표 수집 (댓글 수, 조회수, 스크랩 수 등)
- 머신러닝 모델 학습 (알려진 블로그 지수 데이터 기반)
- 네이버 정책 변경 대응
