## 💡 **왜 이런 문제가 발생했나?**

### **TypeScript + Electron 구조**
1. **소스 코드**: `electron/preload.ts`
2. **빌드 결과**: `electron/preload.js`
3. **UI 폴더**: `electron/ui/` (HTML, CSS, JS)
4. **실행 폴더**: `dist/ui/`

### **복사 전략의 문제**
- `electron/ui/`는 "UI 리소스 폴더"
- TypeScript 빌드 결과는 `electron/`에 생성
- 두 폴더가 분리되어 있어서 수동으로 연결 필요

### **해결 방법**
- 빌드 후 TypeScript 결과를 UI 폴더로 명시적으로 복사
- 이제 항상 최신 버전 사용
