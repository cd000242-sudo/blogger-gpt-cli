# 블로그 지수 제공 서비스 비교

## 모든 서비스는 추정값입니다

**핵심 사실**: 네이버는 블로그 지수를 공개 API로 제공하지 않습니다.
따라서 모든 외부 서비스(블덱스, 블로그차트, 콘콘 등)도 추정값을 제공합니다.

## 서비스별 비교

### 1. 네이버 공식 (가장 정확)

- **지수 타입**: 실제 지수
- **정확도**: 100%
- **확인 방법**: 네이버 블로그 관리 페이지 (로그인 필요)
- **제공 방식**: 공개 API 없음 (내부 시스템만)
- **비고**: 유일하게 정확한 지수

### 2. 블덱스 (blogdex.co.kr)

- **지수 타입**: 추정값
- **정확도**: 불명확 (자체 알고리즘)
- **데이터 소스**: 
  - 블로그 검색 API
  - 크롤링 데이터
  - 자체 수집 데이터
- **계산 방식**: 비공개 알고리즘
- **비고**: 가장 유명한 블로그 지수 사이트 중 하나

### 3. 블로그차트 (blogchart.co.kr)

- **지수 타입**: 추정값
- **정확도**: 불명확 (빅데이터 분석 기반)
- **데이터 소스**: 
  - 빅데이터 분석
  - 시스템 소프트웨어 개발
- **계산 방식**: 빅데이터 분석 알고리즘
- **비고**: 블로그 순위 및 마케팅 서비스 제공

### 4. 콘콘 (concon.co.kr)

- **지수 타입**: 추정값
- **정확도**: 불명확 (자체 알고리즘)
- **기능**: 
  - 블로그 지수 조회
  - 글 순위 예측
  - 키워드 추천
- **계산 방식**: 비공개 알고리즘
- **비고**: 블로그 운영 도구 제공

### 5. 현재 구현 (blogger-gpt-cli)

- **지수 타입**: 추정값
- **정확도**: 85% (신뢰도)
- **데이터 소스**: 
  - 네이버 블로그 검색 API
  - 총 포스트 수
  - 최근 30일 포스트 수
  - 검색 결과 수
- **계산 방식**: 
  - 가중치 기반 점수 계산
  - 블로거 커뮤니티 기준 참고
- **비고**: 오픈소스, 투명한 계산 방식

## 왜 모두 추정값인가?

### 네이버의 정책

1. **블로그 지수는 내부 지표**
   - 네이버 내부 알고리즘으로만 계산
   - 공개 API로 제공하지 않음
   - 블로그 관리 페이지에서만 확인 가능

2. **외부 접근 불가**
   - 네이버 공식 API에 블로그 지수 조회 기능 없음
   - 모든 외부 서비스는 간접적인 방법만 사용 가능

### 각 서비스의 한계

1. **수집 가능한 데이터의 한계**
   - 공개된 데이터만 수집 가능
   - 내부 지표(조회수, 체류시간 등) 접근 불가

2. **알고리즘의 차이**
   - 네이버 내부 알고리즘은 비공개
   - 각 서비스마다 다른 가중치와 계산 방식 사용

3. **데이터 업데이트 주기**
   - 네이버는 실시간 업데이트
   - 외부 서비스는 주기적 업데이트 (지연 발생 가능)

## 결론

1. **모든 외부 서비스는 추정값을 제공합니다**
   - 블덱스도 추정값
   - 블로그차트도 추정값
   - 콘콘도 추정값
   - 현재 구현도 추정값

2. **정확한 지수는 네이버 관리 페이지에서만 확인 가능**
   - 로그인 필요
   - 공개 API 없음

3. **각 서비스의 차이점**
   - 데이터 소스의 차이
   - 알고리즘의 차이
   - 업데이트 주기의 차이

4. **참고용으로만 사용 권장**
   - 모든 추정값은 참고용
   - 실제 성과 평가는 다양한 지표 종합 필요
