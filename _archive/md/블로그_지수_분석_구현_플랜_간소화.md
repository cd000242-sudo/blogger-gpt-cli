# 블로그 지수 분석 기능 - 간소화 플랜

## 📋 요구사항 명확화

**핵심 기능**: 
- 키워드 입력 → 네이버 검색 결과 수집 (순위별)
- 각 블로그의 **네이버 블로그 지수**만 표시

**불필요한 기능**:
- ❌ 복잡한 지수 계산 (네이버에서 제공하는 지수 사용)
- ❌ 상세 정보 크롤링
- ❌ 시각화 차트
- ❌ 리포트 생성

---

## 🎯 최소 기능 정의

### MVP (필수 기능만)
1. 키워드 입력
2. 네이버 블로그 검색 API로 결과 수집 (상위 20-50개)
3. 각 블로그의 네이버 블로그 지수 조회
4. 순위별로 테이블 표시 (순위, 블로그명, 지수)

---

## 📊 네이버 블로그 지수 조회 방법

### 방법 1: 네이버 블로그 API 활용 (우선)
**API 엔드포인트 확인 필요**
- 네이버 블로그 API에서 지수 정보 제공 여부 확인
- 블로그 프로필 API에서 지수 조회 가능한지 확인

### 방법 2: 블로그 프로필 페이지 크롤링
**URL 패턴**: `https://blog.naver.com/{blogId}`
- 블로그 프로필 페이지에서 지수 정보 추출
- HTML 파싱 또는 JavaScript 실행 필요

### 방법 3: 네이버 블로그 지수 공식 확인
- 네이버에서 공개한 지수 계산 공식이 있는지 확인
- 있다면 간단한 계산으로 구현

```javascript
async function analyzeBlogIndex() {
  const keyword = document.getElementById('blogIndexKeyword').value;
  if (!keyword) {
    alert('키워드를 입력해주세요');
    return;
  }
  
  // 로딩 표시
  showLoading();
  
  try {
    const result = await window.electronAPI.invoke('analyze-blog-index', {
      keyword,
      maxResults: 50
    });
    
    if (result.ok) {
      renderBlogIndexTable(result.data);
    } else {
      alert('분석 실패: ' + result.error);
    }
  } catch (error) {
    alert('오류 발생: ' + error.message);
  } finally {
    hideLoading();
  }
}

function renderBlogIndexTable(data) {
  const tbody = document.getElementById('blogIndexTableBody');
  tbody.innerHTML = data.blogs.map(blog => `
    <tr>
      <td>${blog.searchRank}</td>
      <td><a href="${blog.blogUrl}" target="_blank">${blog.blogName}</a></td>
      <td>${blog.blogIndex || 'N/A'}</td>
      <td>${blog.indexRank || '-'}</td>
    </tr>
  `).join('');
}
```

### **Step 5: IPC 핸들러 추가** (1시간)

**파일**: `electron/main.js`

```javascript
ipcMain.handle('analyze-blog-index', async (_event, { keyword, maxResults }) => {
  try {
    const analyzer = await import('../src/utils/blog-index-analyzer');
    const result = await analyzer.analyzeBlogIndex(keyword, maxResults);
    return { ok: true, data: result };
  } catch (error) {
    console.error('[BLOG-INDEX] 분석 실패:', error);
    return { ok: false, error: error.message };
  }
});
```
