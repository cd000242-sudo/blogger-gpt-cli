# ✅ 빠른 확인 체크리스트

## 🎯 일어나시면 이것만 확인하세요!

---

### **1단계: 앱 실행 확인** ⏱️ 10초

```bash
npm start
```

✅ 앱이 켜지나요?  
✅ 오류 팝업이 안 뜨나요?  
✅ 로딩이 완료되나요?

→ **모두 Yes면 성공!** 🎉

---

### **2단계: 핸들러 등록 확인** ⏱️ 5초

터미널에서 다음 메시지를 찾으세요:

```
✅ 모든 IPC 핸들러 등록 완료! (총 80+ 핸들러)
```

→ **보이면 성공!** 🎉

---

### **3단계: 기본 기능 테스트** ⏱️ 1분

#### **환경 설정 열기**
1. 앱에서 "설정" 탭 클릭
2. API 키 입력 필드가 보이나요?

✅ Yes → **정상!**

#### **키워드 검색 (LEWORD)**
1. LEWORD 탭 열기
2. 키워드 입력 → 검색
3. 결과가 나오나요?

✅ Yes → **정상!**

#### **포스트 생성 시도**
1. "포스팅 실행" 탭
2. 주제 입력
3. "생성" 버튼 클릭
4. 오류 없이 진행되나요?

✅ Yes → **완벽!** 🎊

---

## 🚨 문제가 있다면?

### **오류 발생 시**

1. **터미널 확인**
   - 어떤 오류 메시지가 나오나요?
   - `[ERROR]`로 시작하는 줄을 찾으세요

2. **콘솔 확인 (개발자 도구)**
   - F12 키 누르기
   - Console 탭 확인
   - 빨간 오류 메시지 있나요?

3. **로그 파일 확인**
   - 터미널 로그를 복사해서 보내주세요
