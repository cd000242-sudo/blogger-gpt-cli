### **Phase 2: 백엔드 로직** (electron/ui/modules/)

#### 2.1 발행글 목록 관리 (internal-links.js)
```javascript
// 발행글 목록 저장 (localStorage)
function savePublishedPost(post) {
  const posts = getPublishedPosts();
  posts.push(post);
  localStorage.setItem('publishedPosts', JSON.stringify(posts));
}

// 발행글 목록 불러오기
function getPublishedPosts() {
  const posts = localStorage.getItem('publishedPosts');
  return posts ? JSON.parse(posts) : [];
}

// 발행글 목록 모달 표시
function loadPublishedPosts() {
  const posts = getPublishedPosts();
  const modal = document.getElementById('publishedPostsModal');
  const list = document.getElementById('publishedPostsList');
  
  list.innerHTML = posts.map((post, index) => `
    <div class="published-post-item">
      <h3>${post.title}</h3>
      <p>${post.url}</p>
      <button onclick="window.open('${post.url}')">🔗 바로가기</button>
      <button onclick="copyToClipboard('${post.url}')">📋 복사</button>
      <button onclick="selectPost(${index})">✅ 선택</button>
    </div>
  `).join('');
  
  modal.style.display = 'block';
}
```

#### 2.2 내부 링크 거미줄치기 로직
```javascript
let selectedPosts = [];

// 글 선택 (최대 5개)
function selectPost(index) {
  const posts = getPublishedPosts();
  const post = posts[index];
  
  if (selectedPosts.length >= 5) {
    alert('최대 5개까지만 선택할 수 있습니다.');
    return;
  }
  
  if (selectedPosts.find(p => p.url === post.url)) {
    alert('이미 선택한 글입니다.');
    return;
  }
  
  selectedPosts.push(post);
  updateSelectedPostsList();
}

// 선택한 글 목록 업데이트
function updateSelectedPostsList() {
  const list = document.getElementById('selectedPostsList');
  list.innerHTML = selectedPosts.map((post, index) => `
    <div class="selected-post-item">
      <span>${index + 1}. ${post.title}</span>
      <button onclick="removeSelectedPost(${index})">❌</button>
    </div>
  `).join('');
}
```

#### 2.3 관련 글 생성 로직
```javascript
async function generateInternalLinkContent() {
  if (selectedPosts.length === 0) {
    alert('최소 1개 이상의 글을 선택해주세요.');
    return;
  }
  
  // 선택한 글 정보 수집
  const postInfos = selectedPosts.map(post => ({
    title: post.title,
    url: post.url,
    summary: post.summary || ''
  }));
  
  // 백엔드로 전송하여 관련 글 생성
  const result = await window.blogger.generateInternalLinkContent({
    selectedPosts: postInfos,
    platform: 'wordpress' // 또는 'blogger'
  });
  
  // 미리보기 표시
  displayPreview(result.html);
}
```

### **Phase 3: 백엔드 API** (src/core/)

#### 3.1 내부 링크 콘텐츠 생성 함수
```typescript
// src/core/internal-links.ts
export async function generateInternalLinkContent(
  selectedPosts: Array<{ title: string; url: string; summary: string }>,
  geminiKey: string
): Promise<{ html: string; title: string }> {
  
  const prompt = `
당신은 SEO 전문가이자 내부 링크 최적화 전문가입니다.

🎯 **목표**: 다음 ${selectedPosts.length}개의 글을 연결하는 허브 콘텐츠를 작성하세요.

📚 **선택된 글 목록**:
${selectedPosts.map((post, i) => `
${i + 1}. **${post.title}**
   - URL: ${post.url}
   - 요약: ${post.summary}
`).join('\n')}

✅ **작성 요구사항**:
1. **핵심 내용만 간결하게** - 각 글의 핵심을 200-300자로 요약
2. **CTA로 유도** - 각 글로 유도하는 명확한 CTA 포함
3. **내부 링크 최적화** - 자연스러운 앵커 텍스트 사용
4. **SEO 최적화** - 키워드 자연스럽게 포함
5. **가독성** - H2/H3로 구조화, 3-4문단으로 나누기

📝 **출력 형식**:
<h1>제목 (선택한 글들의 공통 주제)</h1>

<h2>1. [첫 번째 글 제목]</h2>
<p>핵심 내용 요약...</p>
<center>
  <a href="${selectedPosts[0].url}" class="internal-link-cta">
    🔗 자세히 보기
  </a>
</center>

<h2>2. [두 번째 글 제목]</h2>
...

⚠️ **절대 금지**:
- 글 내용을 그대로 복사하지 마세요
- 핵심만 간결하게 요약하세요
- CTA는 반드시 <center> 태그 안에 배치하세요
`;

  // Gemini로 생성
  const { GoogleGenerativeAI } = await import('@google/generative-ai');
  const genAI = new GoogleGenerativeAI(geminiKey);
  const model = genAI.getGenerativeModel({ model: 'gemini-2.0-flash-exp' });
  
  const result = await model.generateContent(prompt);
  const html = result.response.text();
  
  // 제목 추출
  const titleMatch = html.match(/<h1>(.*?)<\/h1>/);
  const title = titleMatch ? titleMatch[1] : '관련 글 모음';
  
  return { html, title };
}
```

### **Phase 4: 프롬프트 최적화**

#### 4.1 내부 링크 SEO 전략 프롬프트
```typescript
const INTERNAL_LINK_PROMPT = `
🕸️ **내부 링크 거미줄치기 전략 (SEO 최적화)**

📊 **내부 링크의 중요성**:
- 사이트 전체 SEO 점수 향상
- 크롤링 효율성 증가
- 사용자 체류시간 증가
- 페이지 권위 분산

✅ **내부 링크 최적화 규칙**:

1. **앵커 텍스트 최적화**:
   - ❌ "여기 클릭", "자세히 보기" (SEO 효과 없음)
   - ✅ "연말정산 미리보기 방법", "2025년 세금 환급 가이드" (키워드 포함)

2. **링크 배치 전략**:
   - 본문 상위 30% 이내에 최소 1개
   - 본문 중간에 2-3개
   - 마무리에 1개 (관련 글 추천)

3. **링크 수 제한**:
   - 글당 최대 5-7개 내부 링크
   - 너무 많으면 스팸으로 인식

4. **관련성 확보**:
   - 주제가 관련된 글만 연결
   - 독자에게 실제 도움이 되는 링크만

5. **CTA 디자인**:
   - 눈에 띄되 과하지 않게
   - 클릭 유도 문구 명확하게
   - 중앙 정렬로 강조

🎯 **허브 콘텐츠 전략**:
- 여러 글을 연결하는 중심 글 작성
- 각 글의 핵심만 간결하게 소개
- 독자가 원하는 정보로 빠르게 이동

📝 **작성 예시**:
<h2>연말정산 완벽 가이드</h2>
<p>연말정산을 처음 하시나요? 걱정 마세요! 단계별로 쉽게 설명해드릴게요.</p>

<h3>1단계: 미리보기로 예상 환급액 확인</h3>
<p>먼저 <a href="/post1">2025년 연말정산 미리보기 방법</a>을 통해 예상 환급액을 확인하세요...</p>

<center>
  <a href="/post1" class="internal-link-cta">
    📊 미리보기 방법 자세히 보기
  </a>
</center>
`;
```
