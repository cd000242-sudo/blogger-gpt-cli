# 핵심 요약표 스타일

## 현재 스타일

### 전체 섹션
```css
margin: 60px 0 20px 0;
background: #ffffff;
border-radius: 28px;
padding: 36px 40px;
box-shadow: 0 28px 70px rgba(15, 23, 42, 0.12);
border: 1px solid rgba(226, 232, 240, 0.6);
```

### 제목 (H2)
```css
margin: 0 0 28px 0;
font-size: clamp(1.9rem, 2.2vw + 1rem, 2.3rem);
font-weight: 800;
color: #1a1a1a;  /* 진한 검은색 */
text-align: center;
```

### 테이블 본문 (td)
```css
/* 소제목 (th) */
color: #1a1a1a;  /* 진한 검은색 */

/* 핵심내용 제목 */
color: #1a1a1a;  /* 진한 검은색 */
font-weight: 700;

/* 핵심내용 본문 */
color: #000000;  /* 검은색 */
```

### 테이블
```css
width: 100%;
border-collapse: collapse;
table-layout: fixed;
word-wrap: break-word;
overflow-wrap: break-word;
margin-bottom: 28px;
background: #ffffff;
border-radius: 16px;
overflow: hidden;
border: 1px solid #e2e8f0;
```

### 테이블 헤더 (thead)
```css
background: linear-gradient(135deg, #0ea5e9 0%, #2563eb 100%);
/* 파란색 그라데이션 배경 */
```

### 테이블 헤더 셀 (th)
```css
padding: 18px 24px;
border: none;
color: #ffffff;  /* 흰색 텍스트 */
font-weight: 700;
font-size: clamp(1.08rem, 1.1vw + 0.6rem, 1.18rem);
text-align: left;
word-break: keep-all;
overflow-wrap: break-word;
white-space: normal;
border-bottom: 1px solid rgba(255, 255, 255, 0.25);
```

### 컬럼 너비
- 소제목: 20%
- 핵심내용: 60%
- 관련링크: 20%

### 반응형 디자인
- **768px 이하**: 패딩 축소, 폰트 크기 축소
- **480px 이하**: 더 작은 패딩과 폰트 크기

## 시각적 특징
- ✅ 깔끔한 흰색 배경
- ✅ 파란색 그라데이션 헤더
- ✅ 둥근 모서리 (28px)
- ✅ 부드러운 그림자 효과
- ✅ 반응형 디자인 (모바일 최적화)
