## ✅ 발행 후 확인사항

### PC에서 확인 (Chrome/Firefox 등)

#### 1. 전체 레이아웃
- [ ] 화면을 꽉 채우는지 확인 (좌우 여백 최소화)
- [ ] 본문이 중앙에 배치되어 있는지 확인
- [ ] 사이드바가 오른쪽에 있는지 확인 (있는 경우)

#### 2. 텍스트 크기
- [ ] 제목 (H1): 36-44px 정도로 보이는지 확인
- [ ] 섹션 제목 (H2): 32px 정도로 보이는지 확인
- [ ] 본문 텍스트: 20-22px 정도로 보이는지 확인
- [ ] 텍스트가 읽기 편한 크기인지 확인

#### 3. 여백 및 간격
- [ ] 좌우 여백이 적절한지 확인 (40px 정도)
- [ ] 단락 간격이 적절한지 확인 (24-27px 정도)
- [ ] 섹션 간 여백이 적절한지 확인 (40px 정도)

#### 4. 반응형 디자인
- [ ] 브라우저 창 크기를 조절했을 때 레이아웃이 적절히 조정되는지 확인
- [ ] 태블릿 크기 (1024px 이하)에서도 잘 보이는지 확인

### 모바일에서 확인 (스마트폰 브라우저)

#### 1. 전체 레이아웃
- [ ] 화면을 꽉 채우는지 확인 (좌우 여백 16px 정도)
- [ ] 본문이 전체 너비를 활용하는지 확인
- [ ] 사이드바가 본문 아래로 이동했는지 확인

#### 2. 텍스트 크기
- [ ] 제목 (H1): 30px 정도로 보이는지 확인
- [ ] 섹션 제목 (H2): 24px 정도로 보이는지 확인
- [ ] 본문 텍스트: 17px 정도로 보이는지 확인
- [ ] 모바일에서 읽기 편한 크기인지 확인

#### 3. 여백 및 간격
- [ ] 좌우 여백이 최소화되어 있는지 확인 (16px)
- [ ] 단락 간격이 적절한지 확인 (24px 정도)
- [ ] 섹션 간 여백이 적절한지 확인 (36px 정도)

#### 4. 가독성
- [ ] 텍스트가 너무 작지 않은지 확인
- [ ] 줄간격이 적절한지 확인 (1.8 정도)
- [ ] 단어가 잘리는지 확인 (word-break: keep-all 적용)

## 🔍 CSS 확인 방법

### 개발자 도구로 확인

1. **Chrome/Firefox에서 F12 키 누르기**
2. **Elements 탭에서 확인:**
   - `.max-mode-article` 클래스 찾기
   - `max-width: 100%` 확인
   - `padding: 40px 40px 60px 40px` (PC) 또는 `padding: 32px 16px 48px 16px` (모바일) 확인

3. **Computed 탭에서 확인:**
   - 실제 적용된 `max-width` 값 확인
   - 실제 적용된 `padding` 값 확인
   - 실제 적용된 `font-size` 값 확인

4. **Responsive Design Mode (F12 → Ctrl+Shift+M):**
   - 모바일 크기로 전환하여 확인
   - 다양한 화면 크기에서 테스트
