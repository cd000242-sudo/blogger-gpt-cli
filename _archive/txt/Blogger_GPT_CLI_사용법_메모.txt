Blogger GPT CLI 사용법 메모
========================================

[기본 설정]
1. 앱 실행 후 환경설정 탭으로 이동
2. API 키 입력 (필수):
   - Gemini API: https://makersuite.google.com/app/apikey
   - Pexels API: https://www.pexels.com/api/
   - Google CSE: https://cse.google.com/

[블로그 플랫폼 설정]
WordPress:
- 사이트 URL: https://yoursite.com
- 사용자명: WordPress 관리자 계정
- Application Password: WordPress에서 생성

블로그스팟:
- Blogger ID: 블로그 URL에서 확인
- Google Client ID: Google Cloud Console에서 생성
- Google Client Secret: Google Cloud Console에서 생성
- "블로거 계정 연동" 버튼 클릭

[기본 포스팅 방법]
1. 메인 탭에서 키워드 입력
2.  "포스팅 시작" 버튼 클릭


[썸네일 생성]
텍스트 썸네일:
- 제목과 부제목 입력
- 배경색, 텍스트 색상 선택
- "썸네일 생성" 버튼 클릭

이미지 변환:
- 이미지 파일 업로드
- 썸네일 크기 설정 (1200x630 권장)
- "변환 시작" 버튼 클릭

[엑셀 대량 포스팅]
1. 엑셀 파일 준비 (키워드, 제목, 본문 컬럼 필수)
2. 엑셀 포스팅 탭에서 파일 업로드
3. 데이터 미리보기 및 검증
4. "대량 포스팅 시작" 버튼 클릭

[스케줄 포스팅]
1. 스케줄 관리 탭 선택
2. "새 스케줄" 버튼 클릭
3. 포스팅 콘텐츠 선택
4. 발행 날짜 및 시간 설정
5. "스케줄 저장" 버튼 클릭

[자주 발생하는 문제]

API 키 오류:
- 키가 올바르게 복사되었는지 확인
- 공백이나 특수문자 제거
- API 서비스에서 키 활성화 확인

연동 실패:
WordPress: 사이트 URL, Application Password 확인
블로그스팟: Google Cloud Console OAuth 설정 확인

포스팅 실패:
- 네트워크 연결 상태 확인
- 플랫폼 연동 상태 재확인
- 앱 재시작 후 재시도

[유용한 팁]
- 정기적으로 설정 백업
- API 키 안전하게 보관
- AI 생성 콘텐츠 항상 검토
- 포스팅 전 네트워크 상태 확인
- 정기적으로 앱 업데이트

[지원]
- 이메일: support@bloggergpt.com
- 웹사이트: https://bloggergpt.com
- 커뮤니티: https://community.bloggergpt.com

[주의사항]
- 스케줄 포스팅은 앱이 실행 중이어야 함
- 대량 포스팅 시 API 제한 고려
- 오프라인에서는 AI 기능 사용 불가
- 여러 블로그 관리 시 설정 변경 필요

[빠른 참조]
메인 탭: 키워드 → 제목 생성 → 본문 생성
썸네일 탭: 텍스트/이미지 썸네일 생성
포스팅 탭: 플랫폼 연동 → 포스팅 실행
엑셀 탭: 대량 포스팅 관리
스케줄 탭: 예약 포스팅 관리
환경설정: API 키 및 플랫폼 설정
