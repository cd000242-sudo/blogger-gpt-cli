Blogger GPT CLI - API 키 발급 방법
========================================

[1. Gemini API 키 발급 (필수)]
사이트: https://makersuite.google.com/app/apikey

단계:
1. Google 계정으로 로그인
2. "Create API Key" 버튼 클릭
3. 프로젝트 선택 (기존 또는 새 프로젝트)
4. 생성된 API 키 복사
5. 앱의 "Gemini API" 필드에 붙여넣기

주의사항:
- API 키는 한 번만 표시되므로 안전하게 보관
- 공백이나 특수문자 포함하지 않도록 주의
- Google AI Studio에서 무료 할당량 제공

========================================

[2. Pexels API 키 발급 (이미지용)]
사이트: https://www.pexels.com/api/

단계:
1. Pexels 웹사이트 방문
2. 우측 상단 "Join" 버튼 클릭
3. 이메일로 계정 생성 (무료)
4. 로그인 후 "API" 메뉴 클릭
5. "Request API Key" 버튼 클릭
6. 생성된 API 키 복사
7. 앱의 "Pexels API" 필드에 붙여넣기

무료 할당량:
- 월 200회 요청 (무료)
- 고화질 이미지 다운로드 가능
- 상업적 사용 가능

========================================

[3. Google Custom Search API 키 발급 (검색용)]
사이트: https://cse.google.com/

단계:
1. Google 계정으로 로그인
2. "Add" 버튼으로 새 검색 엔진 생성
3. 검색할 사이트 URL 입력 (예: *.com)
4. 검색 엔진 이름 입력
5. "Create" 버튼 클릭
6. 생성된 검색 엔진 ID 복사 (예: 017576662512468239146:omuauf_lfve)

Google Cloud Console에서 API 키 생성:
1. https://console.cloud.google.com/ 방문
2. 프로젝트 선택 또는 새 프로젝트 생성
3. "API 및 서비스" → "사용 설정된 API" 클릭
4. "Custom Search API" 검색 후 활성화
5. "사용자 인증 정보" → "사용자 인증 정보 만들기" → "API 키"
6. 생성된 API 키 복사
7. 앱의 "Google CSE Key"와 "Google CSE CX" 필드에 입력

무료 할당량:
- 일 100회 검색 (무료)
- 월 10,000회 검색 (유료)

========================================

[4. DALL-E API 키 발급 (선택사항)]
사이트: https://platform.openai.com/api-keys

단계:
1. OpenAI 계정 생성 (유료)
2. 로그인 후 "API Keys" 메뉴 클릭
3. "Create new secret key" 버튼 클릭
4. 키 이름 입력 (예: "Blogger GPT CLI")
5. 생성된 API 키 복사
6. 앱의 "DALL-E API" 필드에 붙여넣기

비용:
- DALL-E 3: $0.040 per image (1024x1024)
- DALL-E 2: $0.020 per image (1024x1024)

========================================

[5. 네이버 API 키 발급 (선택사항)]
사이트: https://developers.naver.com/apps/#/myapps

단계:
1. 네이버 계정으로 로그인
2. "Application 등록" 버튼 클릭
3. 애플리케이션 정보 입력:
   - 애플리케이션 이름: "Blogger GPT CLI"
   - 사용 API: "검색" 선택
   - 서비스 환경: "WEB" 선택
   - 웹 서비스 URL: http://localhost (개발용)
4. "등록" 버튼 클릭
5. 생성된 Client ID와 Client Secret 복사
6. 앱의 "네이버 Customer ID" 필드에 입력

무료 할당량:
- 일 25,000회 요청 (무료)
- 한국어 검색에 최적화

========================================

[API 키 입력 방법]
1. 앱 실행 후 "환경설정" 탭 클릭
2. "API 키 설정" 카드에서 각 필드에 입력:
   - Gemini API: [발급받은 키]
   - Pexels API: [발급받은 키]
   - Google CSE Key: [발급받은 키]
   - Google CSE CX: [검색 엔진 ID]
3. "저장" 버튼 클릭
4. "연동 확인" 버튼으로 테스트

========================================

[문제 해결]
API 키 오류 시:
- 키가 올바르게 복사되었는지 확인
- 공백이나 특수문자 제거
- API 서비스에서 키 활성화 확인
- 할당량 초과 여부 확인

연동 실패 시:
- 네트워크 연결 상태 확인
- API 키 유효성 재확인
- 앱 재시작 후 재시도

========================================

[비용 정리]
무료:
- Gemini API: 월 1,500회 요청
- Pexels API: 월 200회 요청
- Google CSE: 일 100회 검색
- 네이버 API: 일 25,000회 요청

유료:
- DALL-E API: 이미지당 $0.020~$0.040
- Google CSE: 월 10,000회 이후 유료
