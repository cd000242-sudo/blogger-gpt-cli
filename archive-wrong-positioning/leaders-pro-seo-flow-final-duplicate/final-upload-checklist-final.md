# 최종 등록 체크리스트

## 첫 화면

- [ ] 첫 화면이 `글을 계속 쓰는데 블로그 유입 구조가 쌓이지 않는다면`으로 시작한다.
- [ ] 문제를 글 개수가 아니라 키워드와 글의 연결 방식으로 설명한다.
- [ ] LEWORD가 첫 화면에 등장한다.
- [ ] 키워드, 종합글, 하위글, 내부링크, 외부유입 글의 흐름이 보인다.
- [ ] `키워드를 단발성 글감으로 끝내지 않고 종합글·하위글·내부링크·외부유입 글이 연결된 콘텐츠 구조로 확장한다`는 메시지가 들어간다.

## 제목/메인이미지

- [ ] 최종 제목은 `종합글·내부링크·외부유입 글까지 블로그 SEO 구조를 설계합니다`를 사용한다.
- [ ] 최종 메인이미지는 `글은 쓰는데 / 유입 구조가 없다면 / 키워드 · 종합글 · 내부링크 · 외부유입 설계`를 사용한다.
- [ ] 제목과 메인이미지에 결과 확정형 문구가 없다.
- [ ] 작은 화면에서도 핵심 문장이 읽힌다.

## 상세페이지 핵심 섹션

- [ ] 리더스 프로 정의가 블로그 SEO 유입 동선 구축 앱으로 고정되어 있다.
- [ ] 5-LINK 시스템이 들어간다.
- [ ] 현재 제공 툴 3개가 분리되어 있다.
- [ ] 3개 툴이 모두 포함된 올인원 구성임을 명확히 했다.
- [ ] 실제 결과물 예시 섹션이 들어간다.
- [ ] 사용 시나리오 섹션이 들어간다.
- [ ] Before / After가 들어간다.
- [ ] 추천 대상과 비추천 대상이 들어간다.
- [ ] 확장 로드맵은 하단에 짧게 들어간다.

## 패키지

- [ ] 1년 이용권은 400,000원이다.
- [ ] 영구 이용권은 1,500,000원이다.
- [ ] 두 패키지 모두 3개 툴 전체 사용이 포함된다.
- [ ] 기능 차등형이 아니라 이용 기간 차등형으로 설명했다.
- [ ] 1개월 50,000원, 3개월 120,000원은 단기/내부 옵션으로만 언급했다.
- [ ] 설치 가능 PC 수는 [확인 필요]로 남겼다.
- [ ] 계정 수는 [확인 필요]로 남겼다.
- [ ] 키워드/글 생성량은 [확인 필요]로 남겼다.
- [ ] 업데이트 제공 기간 또는 범위는 [확인 필요]로 남겼다.
- [ ] 구매 후 지원 기간은 [확인 필요]로 남겼다.

## 현재 제공/로드맵 분리

- [ ] 현재 제공 기능은 LEWORD, 네이버블로그 자동화, 블로그스팟/워드프레스 자동화만 포함한다.
- [ ] 네이버 카페, 티스토리, 유튜브 영상, 웹툰 기능은 확장 예정으로만 표시한다.
- [ ] 예정 기능이 현재 구매 범위처럼 보이지 않는다.
- [ ] `현재 구매 범위에는 상세페이지에 명시된 제공 기능만 포함됩니다` 문구가 들어간다.
- [ ] 영구 이용권이 모든 향후 기능, 모든 업데이트, 모든 추가 채널을 조건 없이 포함한다는 의미가 아니라고 FAQ에 안내했다.

## 정책 안전

- [ ] 검색/방문/수익 결과를 확정하지 않는다.
- [ ] 플랫폼 판정이나 회피를 암시하지 않는다.
- [ ] 플랫폼 반응을 인위적으로 늘리는 목적을 암시하지 않는다.
- [ ] 품질 낮은 반복 생산 목적처럼 보이지 않는다.
- [ ] `검색 결과, 유입, 수익은 블로그 상태, 콘텐츠 품질, 키워드 경쟁도, 운영 방식에 따라 달라질 수 있습니다` 문구가 들어간다.
- [ ] `플랫폼 정책을 준수하는 범위에서 콘텐츠 운영 흐름을 체계화하는 앱입니다` 문구가 들어간다.

## 구매 전 확인

- [ ] 운영 중인 플랫폼을 묻는다.
- [ ] 사용하려는 계정 수를 묻는다.
- [ ] 원하는 자동화 범위를 묻는다.
- [ ] PC 사용 환경을 묻는다.
- [ ] 세팅 지원 필요 수준을 묻는다.
- [ ] 크몽 메시지 기준으로 상담한다.
