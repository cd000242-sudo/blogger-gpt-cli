# 최종 패키지표

## 패키지 전략

리더스 프로는 기능을 나눠 파는 상품이 아닙니다.

LEWORD 키워드툴, 네이버블로그 자동화, 블로그스팟/워드프레스 자동화 3개 툴을 모두 사용할 수 있는 올인원 패키지입니다.

따라서 크몽 상세페이지에서는 기능 차등형 패키지보다 `이용 기간 차등형`으로 보여주는 것이 더 명확합니다.

| 이용 기간 | 가격 | 비고 |
|---|---:|---|
| 1개월 | 50,000원 | 단기 체험/내부 옵션 |
| 3개월 | 120,000원 | 단기 운영/내부 옵션 |
| 1년 | 400,000원 | 크몽 메인 판매 권장 |
| 영구 이용권 | 1,500,000원 | 크몽 프리미엄 판매 권장 |

- LEWORD 키워드툴
- 네이버블로그 자동화
- 블로그스팟/워드프레스 자동화
- 키워드 기반 SEO 유입 동선 설계
- 글 역할 분배
- 종합글/하위글 구조안
- 내부링크 기본 설계
- 외부유입 글 구성
- 기본 사용 가이드
- 운영 워크플로우 안내
