# FAQ 최종본

## 1. 리더스 프로는 어떤 앱인가요?

리더스 프로는 LEWORD로 키워드를 찾고, 그 키워드를 기준으로 종합글·하위글·내부링크·외부유입 글까지 연결해 블로그 SEO 유입 동선을 구축하는 앱입니다.

단순 키워드툴이나 단순 글쓰기 도구가 아니라, 콘텐츠 운영 흐름을 구조화하는 데 초점을 둡니다.

## 2. 어떤 툴을 사용할 수 있나요?

리더스 프로는 올인원 구성입니다.

1년 이용권과 영구 이용권 모두 LEWORD 키워드툴, 네이버블로그 자동화, 블로그스팟/워드프레스 자동화 3개 툴을 사용할 수 있습니다.

## 3. 1년 이용권에는 무엇이 포함되나요?

1년 이용권은 400,000원입니다.

3개 툴 전체 사용, 키워드 기반 SEO 유입 동선 설계, 글 역할 분배, 종합글/하위글 구조안, 내부링크 기본 설계, 외부유입 글 구성, 기본 사용 가이드가 포함됩니다.

## 4. 영구 이용권에는 무엇이 포함되나요?

영구 이용권은 1,500,000원입니다.

리더스 프로의 현재 제공 기능을 장기적으로 사용하려는 분을 위한 구성입니다.

다만 영구 이용권이 모든 향후 기능, 모든 업데이트, 모든 추가 채널을 조건 없이 포함한다는 의미는 아닙니다.

업데이트 제공 범위, 확장 기능 포함 여부, 구매 후 지원 기간은 구매 전 안내 기준에 따라 적용됩니다.

## 5. 1개월이나 3개월도 있나요?

내부 가격 기준으로 1개월 50,000원, 3개월 120,000원 옵션이 있습니다.

다만 크몽 메인 판매는 1년 이용권과 영구 이용권 중심으로 구성하는 것을 권장합니다.

## 6. LEWORD는 어떤 역할인가요?

LEWORD는 키워드 발굴의 시작점입니다.

중심 키워드와 하위 키워드 후보를 찾고, 어떤 키워드를 종합글이나 하위글로 확장할지 판단하는 데 활용합니다.

## 7. 네이버블로그 자동화는 무엇을 도와주나요?

네이버블로그 콘텐츠 운영에 필요한 반복 작업을 줄이고, 키워드 기반 글 흐름을 더 체계적으로 관리할 수 있도록 돕습니다.

정확한 기능 범위와 사용 조건은 구매 전 확인이 필요합니다.

## 8. 블로그스팟/워드프레스 자동화는 어떤 목적에 쓰나요?

외부유입용 글과 보조 콘텐츠 흐름을 구성하는 데 활용합니다.

종합글과 하위글로 이어지는 외부 콘텐츠 동선을 만드는 역할입니다.

## 9. 결과를 확정적으로 약속하나요?

아닙니다.

검색 결과, 유입, 수익은 블로그 상태, 콘텐츠 품질, 키워드 경쟁도, 운영 방식에 따라 달라질 수 있습니다.

## 10. 어떤 플랫폼을 기준으로 하나요?

현재 제공 기준은 LEWORD 키워드툴, 네이버블로그 자동화, 블로그스팟/워드프레스 자동화입니다.

정확한 지원 범위, 계정 연결 방식, 사용 환경은 구매 전 확인이 필요합니다.

## 11. 예정 기능도 현재 포함되나요?

아닙니다.

네이버 카페, 티스토리, 유튜브 영상, 웹툰 관련 기능은 확장 예정 항목입니다. 현재 구매 범위에는 상세페이지에 명시된 제공 기능만 포함됩니다.

## 12. 내부링크를 어느 정도까지 설계하나요?

종합글과 하위글이 서로 연결되는 기본 흐름을 잡는 방향으로 설명합니다.

정확히 어떤 수준까지 자동화되는지는 실제 기능 확인이 필요합니다.

## 13. 외부유입 글은 어디에 쓰나요?

블로그스팟/워드프레스 기반 보조 콘텐츠로 활용해 종합글·하위글과 연결되는 흐름을 만드는 데 사용합니다.

운영 목적과 플랫폼 정책에 맞는 방식으로 활용해야 합니다.

## 14. 설치가 필요한가요?

앱 기반 상품으로 안내하는 방향입니다.

설치 방식, 사용 가능 OS, 인증 방식은 실제 상품 조건 확인이 필요합니다.

## 15. 구매 전 무엇을 보내야 하나요?

운영 중인 블로그 플랫폼, 사용하려는 계정 수, 원하는 자동화 범위, 필요한 세팅 지원 수준, PC 환경을 보내주세요.

이를 기준으로 1년 이용권과 영구 이용권 중 어떤 구성이 적합한지 안내하는 흐름이 좋습니다.

## 16. 업데이트는 포함되나요?

업데이트 제공 여부와 범위는 구매 전 확인이 필요합니다.

특히 영구 이용권은 앱 사용 기간과 업데이트 제공 범위를 분리해서 안내하는 것을 권장합니다.

## 17. 구매 후 지원 기간은 어떻게 되나요?

지원 기간과 방식은 [확인 필요]입니다.

크몽 메시지를 기준으로 안내하되, 기간과 범위를 명확히 정해두는 것을 권장합니다.

## 18. 어떤 분께 가장 적합한가요?

키워드는 찾지만 콘텐츠 구조가 잡히지 않는 분, 종합글과 하위글의 역할을 나누고 싶은 분, 네이버블로그와 블로그스팟/워드프레스를 함께 운영하고 싶은 분께 적합합니다.

## 19. 어떤 목적에는 맞지 않나요?

운영 없이 자동 결과만 기대하거나, 플랫폼 정책을 벗어난 방식의 자동화를 원하거나, 콘텐츠 품질보다 반복 생산만 원하는 경우에는 적합하지 않습니다.

## 20. 구매 전 상담이 필요한가요?

권장합니다.

앱 상품은 사용 환경, 계정 수, 필요한 자동화 범위에 따라 적합한 안내가 달라질 수 있으므로 구매 전 크몽 메시지로 현재 상황을 보내주세요.
