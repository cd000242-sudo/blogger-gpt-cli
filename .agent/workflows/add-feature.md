## 기능 추가 워크플로우

새로운 기능을 구현하고 기존 시스템에 통합합니다.

### 사전 작업
1. 관련 파일 백업 (자동)
2. 기존 코드 분석

### 작업 순서
3. 필요한 타입/인터페이스 정의

4. 핵심 로직 구현
   - 새 함수/클래스 생성
   - 기존 코드에 통합

5. UI 연동 (필요시)
   - preload.ts에 IPC 핸들러 추가
   - script.js에 UI 로직 추가
   - index.html에 UI 요소 추가

6. 빌드 및 검증
   ```
   npm run build
   ```

7. 기능 테스트

### 완료 조건
- 빌드 성공
- 기능 정상 작동
- 기존 기능에 영향 없음
