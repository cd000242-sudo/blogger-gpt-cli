긴 한글 문구, 가격표, FAQ, 이용 조건은 이미지 생성 모델에 맡기지 않고 HTML/CSS로 렌더링합니다. 아래 프롬프트는 배경 그래픽, 아이콘, 추상 일러스트 제작용입니다.

## 1. Hero 배경

```text
Create a premium B2B SaaS landing page hero background for a Korean blog SEO automation app.

Visual concept:
A single keyword node expands into a structured content network that feels intentionally planned by a human strategist: pillar article, supporting articles, internal links, and external traffic articles.

Style:
Deep navy background, subtle grid, glowing blue and mint connection lines, clean modern dashboard aesthetic, premium automation software product feeling.

No text except abstract UI labels. Do not include people. Do not include search ranking, money, traffic manipulation, or guarantee symbols.
Aspect ratio: vertical landing page section, 1080x1400.
```
