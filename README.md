## 🚀 시스템 소개

이 시스템은 AI를 활용한 블로그 포스팅 자동화 도구로, 다음과 같은 특징을 가지고 있습니다:

- **AI 기반 콘텐츠 생성**: Gemini 2.0을 활용한 고품질 콘텐츠 자동 생성
- **다양한 콘텐츠 모드**: SEO 최적화, 애드센스 승인, 페러프레이징 등 6가지 전문 모드
- **썸네일 자동 생성**: 텍스트 기반 및 AI 이미지 생성 지원
- **수익화 최적화**: 애드센스, 아피필리에이트 수익화에 특화
- **다중 플랫폼 지원**: WordPress, Blogger 지원

## 🎯 주요 기능

### 콘텐츠 생성
- **자동 제목 생성**: SEO 최적화된 클릭률 높은 제목
- **본문 자동 생성**: 5개 섹션으로 구성된 완성도 높은 글
- **키워드 최적화**: 자연스러운 키워드 배치로 검색엔진 최적화
- **이미지 자동 삽입**: H2 섹션별 이미지 자동 생성 및 삽입

### 수익화 기능
- **애드센스 최적화**: E-E-A-T 강화로 승인율 향상
- **CTA 자동 삽입**: 구매 전환율 향상
- **내부 링크 최적화**: 블로그 체류시간 증가
- **소셜 미디어 공유**: 트래픽 증가 유도

### 썸네일 시스템
- **텍스트 기반 생성**: 커스텀 텍스트로 전문적 썸네일
- **AI 배경 생성**: DALL-E, Pexels 연동
- **외부 이미지 최적화**: URL 기반 이미지 크기 자동 조정
- **자동 적용**: 생성 즉시 포스팅에 적용

## 📝 콘텐츠 모드별 특징

### 1. **SEO 최적화 모드** (External) - ⭐ 추천
- **목적**: 검색엔진 상위 노출 최적화
- **글자수**: 4,200자 (서론 600자 + 본론 3,600자 + 결론 1,000자)
- **섹션 구조**:
  - 서론: 독자 관심 유도 + 가치 제시
  - [주제]란 무엇인가?: 기초 개념 + 특징 설명
  - [주제]의 특징과 장점: 비교 분석 + 선택 기준
  - [주제] 활용 방법과 사례: 실전 가이드 + 성공 사례
  - 결론 및 요약: 핵심 정리 + 추가 정보 공유
- **장점**: SEO 최적화 + 정보 공유 + 전환율 향상
- **추천 사용**: 일반 블로그 포스팅, 정보 공유형 콘텐츠

### 2. **애드센스 승인 모드** (Adsense)
- **목적**: Google 애드센스 승인 최적화
- **글자수**: 6,000자 (7개 섹션)
- **섹션 구조**:
  - 작성자 소개: 전문성 어필
  - 서론: 문제 공감 + 가치 제시
  - 완전 이해: 교육적 가치 제공
  - 직접 경험: 독창성 강조
  - 실행 가이드: 실용적 가치
  - 비교 분석: 객관성 강화
  - 마무리: 신뢰성 구축
- **장점**: E-E-A-T 강화, 승인율 90% 이상
- **추천 사용**: 애드센스 승인을 목표로 하는 블로그

### 3. **일관성/거미줄치기 모드** (Spiderwebbing)
- **목적**: 체계적이고 일관성 있는 정보 제공
- **글자수**: 5,600자 (5개 섹션)
- **섹션 구조**:
  - 서론: 검색 의도 파악
  - 기본 개념: 정의 + 중요성
  - 종류 분류: 유형별 특징 + 선택 기준
  - 실전 가이드: 단계별 실행 방법
  - 비교 추천: 객관적 평가
  - FAQ: 추가 질문 해소
- **장점**: 정보의 체계성, 사용자 만족도 높음
- **추천 사용**: 튜토리얼, 가이드 콘텐츠

### 4. **페러프레이징 모드** (Paraphrasing)
- **목적**: 기존 콘텐츠를 완전히 다른 표현으로 재작성
- **글자수**: 4,400자
- **섹션 구조**: 원문과 다른 구조로 재구성
- **장점**: 중복 콘텐츠 방지, SEO 다양성
- **추천 사용**: 기존 콘텐츠 리뉴얼, 콘텐츠 확장

### 5. **내부 링크 최적화 모드** (Internal)
- **목적**: 블로그 내부 체류시간 증가
- **특징**: 관련 콘텐츠 자동 링크 삽입
- **장점**: 이탈률 감소, 체류시간 증가
- **추천 사용**: 시리즈 콘텐츠, 관련 주제 연계

### 6. **쇼핑 모드** (Shopping)
- **목적**: 제품 판매 전환율 극대화
- **글자수**: 8,000자 (AIDA 모델 기반)
- **섹션 구조**:
  - Attention: 관심 유도
  - Interest: 제품 가치 설명
  - Desire: 구매 욕구 강화
  - Action: 즉시 구매 유도
- **장점**: 판매 전환율 300% 향상
- **추천 사용**: 제품 리뷰, 판매 페이지

## 🖼️ 썸네일 생성기

### 텍스트 기반 썸네일
1. **텍스트 입력**: 썸네일에 표시할 텍스트 입력
2. **폰트 크기 조절**: 40px ~ 600px 범위에서 조절
3. **색상 설정**: 텍스트/배경색 커스터마이징
4. **배경 옵션**:
   - **단색**: 기본 그라데이션 배경
   - **AI 생성**: 프롬프트로 AI 배경 생성
   - **외부 이미지**: URL로 배경 이미지 적용

### 고급 기능
- **AI 배경 생성**: "우주 배경", "산 풍경" 등 프롬프트로 생성
- **이미지 최적화**: 외부 이미지를 1200x630으로 자동 크롭/리사이즈
- **자동 적용**: 생성 즉시 포스팅에 적용

## ⚙️ 설정 및 사용법

### 기본 설정
1. **API 키 설정**: Gemini, OpenAI, Pexels 등 API 키 입력
2. **블로그 설정**: WordPress/Blogger 연결 정보
3. **콘텐츠 설정**: 기본 톤, 글자수, 키워드 등

### 콘텐츠 생성 절차
1. **주제 입력**: 메인 키워드 및 관련 키워드 입력
2. **모드 선택**: 목적에 맞는 콘텐츠 모드 선택
3. **설정 조정**: 세부 옵션 (톤, 길이, 이미지 등)
4. **미리보기**: 생성 전 내용 확인
5. **발행**: 자동 또는 예약 발행

### 썸네일 생성 절차
1. **텍스트 입력**: 썸네일 문구 입력
2. **스타일 설정**: 색상, 폰트, 배경 선택
3. **생성 및 다운로드**: PNG 파일로 저장
4. **자동 적용**: 포스팅 시 자동 사용

## 🔧 문제 해결

### 텍스트가 안 보이는 문제
- **원인**: 그라데이션 텍스트 효과로 인해 대비가 부족할 수 있음
- **해결**: 스킨에서 --text-supreme 변수를 명확한 색상으로 설정

### 썸네일이 적용되지 않는 문제
- **원인**: localStorage에 저장되지 않았거나 payload에 포함되지 않음
- **해결**: 썸네일 생성 후 "썸네일 사용하기" 버튼을 반드시 클릭

### 콘텐츠 생성이 느린 문제
- **원인**: 순차적 처리로 인한 대기 시간
- **해결**: 안정성을 위해 의도적으로 순차 처리 중 (충돌 방지)

### API 할당량 초과
- **원인**: 동시에 많은 요청 발생
- **해결**: 재시도 및 폴백 모델 자동 전환 지원
