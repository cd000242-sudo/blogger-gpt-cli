# FAQ 최종본

## 1. 리더스 프로는 어떤 앱인가요?

리더스 프로는 LEWORD로 키워드를 찾고, 그 키워드를 기준으로 종합글·하위글·내부링크·외부유입 글까지 연결해 블로그 SEO 유입 동선을 구축하는 올인원 앱입니다.

단순 키워드툴이나 단순 글쓰기 도구가 아니라, 콘텐츠 운영 흐름을 구조화하는 데 초점을 둡니다.

## 2. 어떤 툴을 사용할 수 있나요?

현재 제공되는 툴은 3개입니다.

| 툴 | 역할 |
|---|---|
| LEWORD 키워드툴 | 키워드 발굴 |
| 네이버블로그 자동화 | 네이버블로그 콘텐츠 운영 보조 |
| 블로그스팟/워드프레스 자동화 | 외부유입용 보조 콘텐츠 구성 |

1년 이용권과 영구 이용권 모두 3개 툴을 사용할 수 있습니다.

## 3. 1년 이용권에는 무엇이 포함되나요?

1년 이용권은 400,000원입니다.

현재 제공 툴 3개 전체 사용, 키워드 기반 SEO 유입 동선 설계, 글 역할 분배, 종합글/하위글 구조안, 내부링크 기본 설계, 외부유입 글 구성, 기본 사용 가이드, 운영 워크플로우 안내가 포함됩니다.

## 4. 영구 이용권에는 무엇이 포함되나요?

영구 이용권은 1,500,000원입니다.

리더스 프로의 현재 제공 기능을 장기적으로 사용하려는 분을 위한 구성입니다.

다만 영구 이용권이 모든 향후 기능, 모든 업데이트, 모든 추가 채널을 조건 없이 포함한다는 의미는 아닙니다.

## 5. 1개월이나 3개월도 있나요?

내부 가격 기준으로 1개월 50,000원, 3개월 120,000원 옵션이 있습니다.

다만 크몽 메인 판매는 1년 이용권과 영구 이용권 중심으로 구성합니다.

## 6. Windows에서 사용할 수 있나요?

네. 리더스 프로는 Windows 기준으로 안내합니다.

구매 전 사용 중인 PC 환경을 크몽 메시지로 보내주시면 확인 후 안내드리겠습니다.

## 7. Mac에서도 사용할 수 있나요?

Mac은 테스트 후 사용 가능 여부를 별도로 안내드립니다.

Mac 사용자는 구매 전 반드시 사용 환경을 먼저 보내주세요.

## 8. 설치 가능한 PC 수 제한이 있나요?

설치 가능 PC 수에는 별도 제한을 두지 않습니다.

다만 동일 아이디/비밀번호의 중복 로그인은 불가합니다.

## 9. 라이선스는 어떻게 등록하나요?

아이디/비밀번호를 입력한 뒤 라이선스코드를 등록하는 방식입니다.

구매 후 기본 사용 가이드에 따라 진행하시면 됩니다.

## 10. 여러 계정을 사용할 수 있나요?

계정 수에는 별도 제한을 두지 않습니다.

다만 안정적인 운영을 위해 최대 5개 사용을 권장합니다.

## 11. 글 생성량 제한이 있나요?

글 생성량에는 별도 제한을 두지 않습니다.

다만 플랫폼 정책과 콘텐츠 품질을 고려해 정상적인 운영 범위에서 사용하는 것을 권장합니다.

## 12. 업데이트는 포함되나요?

구매 후 1년간 무상 업데이트가 제공됩니다.

1년 이후 업데이트 유지를 원할 경우 월 30,000원 기준으로 안내됩니다.

## 13. 구매 후 문의 지원은 어떻게 되나요?

기본 사용 문의는 구매 후 15일간 지원합니다.

신규 기능 개발, 개별 커스텀, 운영 대행, 별도 세팅 작업은 기본 문의 지원과 구분됩니다.

## 14. LEWORD는 어떤 역할인가요?

LEWORD는 키워드 발굴의 시작점입니다.

중심 키워드와 하위 키워드 후보를 찾고, 어떤 키워드를 종합글이나 하위글로 확장할지 판단하는 데 활용합니다.

## 15. 네이버블로그 자동화는 무엇을 도와주나요?

네이버블로그 콘텐츠 운영에 필요한 반복 작업을 줄이고, 키워드 기반 글 흐름을 더 체계적으로 관리할 수 있도록 돕습니다.

플랫폼 정책을 준수하는 범위에서 콘텐츠 운영 흐름을 체계화하는 용도로 사용하는 것을 권장합니다.

## 16. 블로그스팟/워드프레스 자동화는 어떤 목적에 쓰나요?

외부유입용 글과 보조 콘텐츠 흐름을 구성하는 데 활용합니다.

종합글과 하위글로 이어지는 외부 콘텐츠 동선을 만드는 역할입니다.

## 17. 결과를 확정적으로 약속하나요?

아닙니다.

검색 결과, 유입, 수익은 블로그 상태, 콘텐츠 품질, 키워드 경쟁도, 운영 방식에 따라 달라질 수 있습니다.

리더스 프로는 결과를 확정하는 상품이 아니라 콘텐츠 운영 구조를 체계화하는 앱입니다.

## 18. 예정 기능도 현재 포함되나요?

아닙니다.

네이버 카페, 티스토리, 유튜브 영상, 웹툰 관련 기능은 확장 예정 항목입니다.

현재 구매 범위에는 상세페이지에 명시된 제공 기능만 포함됩니다.

## 19. 어떤 분께 적합한가요?

키워드는 찾지만 콘텐츠 구조가 잡히지 않는 분, 종합글과 하위글의 역할을 나누고 싶은 분, 네이버블로그와 블로그스팟/워드프레스를 함께 운영하고 싶은 분께 적합합니다.

## 20. 어떤 목적에는 맞지 않나요?

운영 없이 자동 결과만 기대하거나, 플랫폼 정책을 벗어난 방식의 자동화를 원하거나, 콘텐츠 품질보다 반복 생산만 원하는 경우에는 적합하지 않습니다.

## 21. 구매 전 무엇을 보내야 하나요?

구매 전 크몽 메시지로 아래 내용을 보내주세요.

- 사용 중인 PC 환경
- 운영 중인 블로그 플랫폼
- 사용하려는 계정 수
- 필요한 기능 범위
- 키워드 발굴만 필요한지, 블로그 자동화까지 필요한지

확인 후 1년 이용권과 영구 이용권 중 어떤 구성이 적합한지 안내드리겠습니다.
