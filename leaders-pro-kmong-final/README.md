# 리더스 프로 크몽 등록 최종본

이 폴더는 리더스 프로를 크몽에 등록할 때 사용할 최종 문서만 모아둔 폴더입니다.

새로운 기획을 추가하지 않고, 확정된 상품 조건과 가격 기준만 반영했습니다.

## 최종 상품 정의

리더스 프로는 LEWORD 키워드툴을 시작점으로 네이버블로그, 블로그스팟, 워드프레스 콘텐츠 흐름을 연결해 블로그 SEO 유입 구조를 설계하는 올인원 앱입니다.

현재 제공되는 툴은 아래 3개입니다.

| 구분 | 제공 툴 |
|---|---|
| 1 | LEWORD 키워드툴 |
| 2 | 네이버블로그 자동화 |
| 3 | 블로그스팟/워드프레스 자동화 |

## 최종 제목

종합글·내부링크·외부유입 글까지 블로그 SEO 구조를 설계합니다

## 최종 패키지

리더스 프로는 기능을 나눠 판매하지 않는 올인원 패키지입니다.

| 패키지 | 가격 | 구성 |
|---|---:|---|
| 1년 이용권 | 400,000원 | 현재 제공 툴 3개 전체 사용 |
| 영구 이용권 | 1,500,000원 | 현재 제공 툴 3개 장기 사용 |

내부 옵션으로 1개월 50,000원, 3개월 120,000원 기준이 있으나 크몽 메인 상세페이지에서는 1년 이용권과 영구 이용권 중심으로 안내합니다.

## 핵심 구조

리더스 프로는 아래 5-LINK 흐름으로 설명합니다.

1. Keyword: LEWORD로 유입 가능 키워드 후보 찾기
2. Role: 키워드별 글 역할 분배
3. Pillar: 전체 주제를 묶는 종합글 구조 설계
4. Link: 종합글과 하위글 내부링크 연결
5. Traffic: 블로그스팟/워드프레스 기반 외부유입 글 구성

## 등록 기준

상세페이지에는 성과를 확정하는 표현, 플랫폼 정책 회피를 암시하는 표현, 조건 없는 지원 또는 모든 향후 기능 포함처럼 오해될 수 있는 표현을 넣지 않습니다.

결과는 블로그 상태, 콘텐츠 품질, 키워드 경쟁도, 운영 방식에 따라 달라질 수 있음을 명확히 안내합니다.
