# 크몽 등록 전 상품 조건 확정본

이 문서는 리더스 프로 크몽 등록 문서에 반영할 실제 상품 조건의 최종 기준입니다.

## 최종 확정값

| 항목 | 최종값 |
|---|---|
| 사용 가능 OS | Windows 기준 |
| Mac 사용 | 테스트 후 가능 여부 별도 안내 |
| 설치 가능 PC 수 | 제한 없음 |
| 라이선스 인증 방식 | 아이디/비밀번호 입력 후 라이선스코드 등록 |
| 중복 로그인 | 동일 아이디/비밀번호 중복 로그인 불가 |
| 계정 수 제한 | 별도 제한 없음 |
| 권장 계정 수 | 안정 운영을 위해 최대 5개 권장 |
| 키워드/글 생성량 제한 | 글 생성량에는 별도 제한 없음 |
| 업데이트 제공 범위 | 구매 후 1년 무상 업데이트 |
| 1년 이후 업데이트 | 월 30,000원 |
| 구매 후 지원 기간 | 기본 사용 문의 15일 지원 |
| 지원 제외 또는 별도 협의 | 신규 기능 개발, 개별 커스텀, 운영 대행, 별도 세팅 작업 |

## 패키지 기준

| 패키지 | 가격 | 기준 |
|---|---:|---|
| 1년 이용권 | 400,000원 | 크몽 메인 판매 |
| 영구 이용권 | 1,500,000원 | 크몽 프리미엄 판매 |

내부 옵션:

| 옵션 | 가격 | 사용 방식 |
|---|---:|---|
| 1개월 | 50,000원 | 메인 노출보다 상담용 옵션 |
| 3개월 | 120,000원 | 메인 노출보다 상담용 옵션 |

## 구매 전 상담에서 확인할 내용

- 사용 중인 PC 환경
- 운영 중인 블로그 플랫폼
- 사용하려는 계정 수
- 필요한 기능 범위
- 키워드 발굴만 필요한지, 블로그 자동화까지 필요한지
- 세팅 지원 또는 사용 가이드가 필요한 수준

## 문구 기준

영구 이용권은 현재 제공 기능을 장기적으로 사용할 수 있는 구성입니다.

모든 향후 기능, 모든 업데이트, 모든 추가 채널이 조건 없이 포함된다는 의미로 안내하지 않습니다.
