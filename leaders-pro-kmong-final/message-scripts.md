# 구매 전 상담 스크립트 최종본

## 스크립트 1. 첫 문의 응답

안녕하세요. 리더스 프로입니다.

문의 주셔서 감사합니다.

리더스 프로는 LEWORD 키워드툴, 네이버블로그 자동화, 블로그스팟/워드프레스 자동화를 통해 키워드 → 종합글/하위글 → 내부링크 → 외부유입 글까지 이어지는 블로그 SEO 유입 동선을 구축하는 올인원 앱입니다.

정확한 안내를 위해 아래 5가지만 보내주세요.

1. 사용 중인 PC 환경
2. 운영 중인 블로그 플랫폼
3. 사용하려는 계정 수
4. 필요한 기능 범위
5. 키워드 발굴만 필요한지, 블로그 자동화까지 필요한지

확인 후 1년 이용권과 영구 이용권 중 어떤 구성이 적합한지 안내드리겠습니다.

## 스크립트 2. 상품 정의 설명

리더스 프로는 단순 키워드툴이나 단순 글쓰기 도구가 아닙니다.

LEWORD로 키워드를 찾고, 그 키워드를 기준으로 글 역할을 나누고, 종합글·하위글·내부링크·외부유입 글까지 이어지는 블로그 SEO 유입 동선을 만드는 앱입니다.

## 스크립트 3. 이용권 안내

리더스 프로는 기능을 나눠 판매하지 않는 올인원 구성입니다.

1년 이용권과 영구 이용권 모두 LEWORD 키워드툴, 네이버블로그 자동화, 블로그스팟/워드프레스 자동화 3개 툴을 사용할 수 있습니다.

1년 이용권은 400,000원이고, 영구 이용권은 1,500,000원입니다.

## 스크립트 4. 단기 이용권 문의 응답

내부 가격 기준으로 1개월 50,000원, 3개월 120,000원 옵션도 있습니다.

다만 크몽 메인 판매 구성은 1년 이용권과 영구 이용권 중심으로 안내하고 있습니다.

짧게 사용해보고 싶은 경우에는 먼저 사용 목적과 필요한 기능 범위를 확인한 뒤 안내드리겠습니다.

## 스크립트 5. 사용 환경 문의 응답

리더스 프로는 Windows 기준으로 안내합니다.

Mac 사용은 테스트 후 가능 여부를 별도로 안내드리므로, Mac 사용자는 구매 전 환경을 먼저 보내주세요.

설치 가능 PC 수에는 별도 제한을 두지 않지만, 동일 아이디/비밀번호 중복 로그인은 불가합니다.

## 스크립트 6. 계정/글 생성량 문의 응답

계정 수에는 별도 제한을 두지 않습니다.

다만 안정적인 운영을 위해 최대 5개 사용을 권장합니다.

글 생성량에도 별도 제한은 없지만, 플랫폼 정책과 콘텐츠 품질을 고려해 정상적인 운영 범위에서 사용하는 것을 권장합니다.

## 스크립트 7. 업데이트/지원 문의 응답

구매 후 1년간 무상 업데이트가 제공됩니다.

1년 이후 업데이트 유지를 원할 경우 월 30,000원 기준으로 안내됩니다.

기본 사용 문의는 구매 후 15일간 지원합니다.

신규 기능 개발, 개별 커스텀, 운영 대행, 별도 세팅 작업은 기본 문의 지원과 구분됩니다.

## 스크립트 8. 결과 문의 응답

리더스 프로는 콘텐츠 구조와 운영 흐름을 돕는 앱입니다.

검색 결과, 유입, 수익은 블로그 상태, 콘텐츠 품질, 키워드 경쟁도, 운영 방식에 따라 달라질 수 있습니다.

구매 전 사용 목적과 운영 환경을 확인한 뒤 진행하는 것을 권장드립니다.

## 스크립트 9. 예정 기능 문의 응답

네이버 카페, 티스토리, 유튜브 영상, 웹툰 관련 기능은 확장 예정 항목입니다.

현재 구매 범위에는 상세페이지에 명시된 제공 기능만 포함됩니다.

예정 기능은 개발 및 제공 범위가 확정되는 대로 별도 안내됩니다.
