# 크몽 등록 순서

## 1. 제목 입력

아래 제목을 그대로 사용합니다.

종합글·내부링크·외부유입 글까지 블로그 SEO 구조를 설계합니다

## 2. 메인이미지 등록

`kmong-main-image-spec.md` 기준으로 제작한 이미지를 등록합니다.

메인이미지 문구:

글은 쓰는데  
유입 구조가 없다면

키워드 · 종합글 · 내부링크 · 외부유입 설계

## 3. 짧은 소개 입력

LEWORD 키워드툴, 네이버블로그 자동화, 블로그스팟/워드프레스 자동화를 하나로 연결해 블로그 콘텐츠 운영 구조를 설계하는 올인원 앱입니다.

## 4. 상세페이지 본문 입력

`kmong-copy-paste-final.md`의 상세페이지 본문을 순서대로 붙여넣습니다.

## 5. 패키지 입력

`package-table-final.md` 기준으로 아래 2개 패키지만 메인에 노출합니다.

| 패키지 | 가격 |
|---|---:|
| 1년 이용권 | 400,000원 |
| 영구 이용권 | 1,500,000원 |

1개월 50,000원, 3개월 120,000원은 내부/상담용 옵션으로만 사용합니다.

## 6. FAQ 입력

`faq-final.md`의 FAQ를 등록합니다.

OS, PC 수, 라이선스, 계정 수, 글 생성량, 업데이트, 구매 후 지원 조건을 FAQ에 반드시 포함합니다.

## 7. 구매 전 메시지 안내 입력

구매 전 크몽 메시지로 아래 내용을 보내달라고 안내합니다.

- 사용 중인 PC 환경
- 운영 중인 블로그 플랫폼
- 사용하려는 계정 수
- 필요한 기능 범위
- 키워드 발굴만 필요한지, 블로그 자동화까지 필요한지

## 8. 마지막 위험 문구 확인

등록 직전 `final-risk-check-report.md`를 확인합니다.

성과를 확정하는 표현, 플랫폼 정책 회피를 암시하는 표현, 조건 없는 지원처럼 보일 수 있는 표현이 없는지 마지막으로 확인합니다.
