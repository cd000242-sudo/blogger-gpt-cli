| 항목 | 최종값 |
|---|---|
| 사용 가능 OS | Windows 기준 |
| Mac 사용 | 테스트 후 가능 여부 별도 안내 |
| 설치 가능 PC 수 | 제한 없음 |
| 라이선스 인증 방식 | 아이디/비밀번호 입력 후 라이선스코드 등록 |
| 중복 로그인 | 동일 아이디/비밀번호 중복 로그인 불가 |
| 계정 수 제한 | 별도 제한 없음 |
| 권장 계정 수 | 안정 운영을 위해 최대 5개 권장 |
| 글 생성량 | 별도 제한 없음 |
| 업데이트 | 구매 후 1년 무상 |
| 1년 이후 업데이트 | 월 30,000원 |
| 구매 후 문의 지원 | 기본 사용 문의 15일 지원 |
| 별도 협의 범위 | 신규 기능 개발, 개별 커스텀, 운영 대행, 별도 세팅 작업 |

## 등록 문구 기준

- 현재 제공 기능은 LEWORD 키워드툴, 네이버블로그 자동화, 블로그스팟/워드프레스 자동화 3개입니다.
- 1년 이용권과 영구 이용권 모두 3개 툴 전체 사용이 포함됩니다.
- 영구 이용권은 현재 제공 기능을 장기적으로 사용하기 위한 구성입니다.
- 예정 기능은 현재 구매 범위에 포함된 기능처럼 쓰지 않습니다.
- 결과는 블로그 상태, 콘텐츠 품질, 키워드 경쟁도, 운영 방식에 따라 달라질 수 있음을 안내합니다.
