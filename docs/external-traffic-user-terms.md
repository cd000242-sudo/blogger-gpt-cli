# 외부유입 글 생성 — 사용자 약관 (베타)

**버전**: 2026-06-01-beta
**상태**: 베타 — 외부 법무 자문 미완료
**적용 범위**: LEADERNAM Orbit 앱의 "외부유입 글 생성" 기능

---

## 1. 목적 및 정의

본 약관("**약관**")은 LEADERNAM Orbit("**본 도구**")의 "외부유입 글 생성" 기능("**기능**")을 사용하는 사용자("**사용자**")가 본 도구 및 기능을 통해 생성된 결과물을 사용함에 있어 책임과 한계를 명시합니다.

- "**외부 채널**": 인스타그램·X·페이스북·디시인사이드·네이버 카페 등 본 도구가 변환 결과를 생성하는 대상 플랫폼.
- "**변환 결과**": 본 도구가 사용자의 블로그 글을 외부 채널에 맞춰 LLM으로 변환한 텍스트·이미지 자리·해시태그 등 일체.

## 2. 도구의 한계 (가장 중요)

> ⚠️ 본 도구는 ban·정지·차단 위험을 줄이는 **보조 장치**이며, **제거하지 않습니다.**

1. 본 도구는 외부 채널의 정책 변경·운영진 재량·알고리즘 변화를 실시간으로 추적하지 못합니다.
2. 위험도 평가(`riskScore`)는 패턴 기반 추정이며 실제 안전성을 보장하지 않습니다.
3. 본 도구가 "안전 추정(🟢)"으로 표시한 결과도 외부 채널에서 정지·차단될 수 있습니다.

## 3. 사용자 책임

사용자는 본 기능을 사용함으로써 다음을 인지하고 동의합니다.

1. **각 외부 채널의 자체 규정·운영진 재량·법규(표시광고법·약관규제법 등) 준수 책임은 사용자에게 있습니다.**
2. 본 도구의 변환 결과를 게시함으로써 발생한 모든 법적·운영적 책임(정지·차단·IP 차단·소송 등)은 사용자가 부담합니다.
3. 일부 외부 채널(디시인사이드·더쿠·루리웹 등)은 자기 홍보를 **명시적으로 금지**합니다. 해당 채널에 게시할 경우 사용자는 위 정책을 인지하고 자기 책임으로 진행하는 것입니다.
4. 본 도구는 외부 채널의 ToS 위반을 권장하지 않습니다. 사용자가 ToS를 위반하면서 결과를 게시한 경우 본 도구는 책임지지 않습니다.

## 4. 사용 로그

1. 본 도구는 사용 로그(생성 시점·채널·위험 등급·동의 기록)를 **로컬에만** 보존합니다.
2. 사용 로그는 분쟁 발생 시 사용자가 본인 위험 인지를 입증할 자료로 활용될 수 있습니다.
3. **사용 로그는 양면성이 있습니다.**
   - ✅ 사용자가 위험을 인지하고 동의한 기록 — 정지·차단 분쟁 시 본인 입증 자료
   - ⚠️ 사용자가 위험을 알면서 게시했다는 증거 — 외부 채널이 사용자에게 불리한 자료로 활용 가능
4. 사용자는 언제든 사용 로그를 삭제할 수 있으며, 본 도구는 90일 이후 동의 기록 외 자동 삭제합니다.

## 5. 개인정보·데이터 보안

1. 본 도구는 사용자의 글·블로그 URL·결과를 **외부 서버로 전송하지 않습니다.**
   - 예외: LLM 호출 시 Gemini(또는 사용자 설정 시 OpenAI/Claude) API에 입력이 전달됩니다. 해당 API 제공자의 정책이 적용됩니다.
2. 로컬 데이터(consent·사용 로그·피드백·사용자 패턴)는 OS 키체인(`safeStorage`) 기반 AES 암호화로 보존합니다.
3. OS 키체인 미사용 환경에서는 평문 저장될 수 있으며, 본 도구가 사용자에게 명시적으로 경고합니다.

## 6. 익명 협력 풀 (옵트인)

1. 사용자가 "협력 풀 참여"를 명시 선택한 경우에만 다음 익명화 데이터가 수집됩니다.
   - 채널 ID, 변환 시점, 위험 등급(band), 사용자 verdict(👍/👎/정지)
   - **포함하지 않는 것**: 사용자 ID·블로그 URL·원문·결과 원문
2. 사용자는 언제든 협력 풀 참여를 철회할 수 있으며, 본 도구는 즉시 전송을 중단합니다.
3. 참여 시 사용자는 confidence 'community-validated' 패턴 우선 노출 등의 혜택을 받습니다.

## 7. 면책

1. 본 도구는 사용자가 외부 채널의 자체 규정·법규를 위반함으로써 발생한 손해에 대해 어떠한 책임도 지지 않습니다.
2. 본 도구는 변환 결과의 **정확성·안전성·도달·수익 발생을 보장하지 않습니다.**
3. 본 도구는 외부 채널의 정책 변경에 따른 결과 부정합·정지·차단에 대해 책임지지 않습니다.
4. 본 도구는 LLM 제공자(Gemini·OpenAI·Claude)의 정책 변경·요금 변경·서비스 중단에 따른 손해를 책임지지 않습니다.
5. 본 도구는 "베타" 상태이며, **외부 법무 자문이 미완료**된 상태로 제공됩니다. 사용자는 이를 인지하고 베타 사용을 진행합니다.

## 8. 약관 변경

1. 본 도구는 약관을 변경할 수 있으며, 변경 시 사용자에게 재동의를 요청합니다.
2. 사용자가 재동의하지 않을 경우 본 기능을 사용할 수 없습니다.

## 9. 준거법·관할

1. 본 약관의 해석 및 준거법은 대한민국 법령을 따릅니다.
2. 본 약관과 관련된 분쟁은 사용자의 주소지 관할 법원을 1심 관할로 합니다.

- **약관규제법**: 본 약관의 일부 조항이 일방적 면책으로 해석될 경우 무효일 수 있습니다.
- **부정경쟁방지법**: 익명·위장 후킹 패턴 사용은 사용자 책임이며, 본 도구는 의도적 위장을 권장하지 않습니다.
