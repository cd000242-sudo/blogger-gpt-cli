# 외부유입 글 생성 — 면책 조항 (베타)

**버전**: 2026-06-01-beta
**상태**: 베타 — 외부 법무 자문 미완료

---

## 한 줄 면책

> **이 도구는 ban 위험을 줄이는 보조 장치이지, 제거하는 마법이 아닙니다.**
> **운영 노하우가 부족한 사용자는 본 도구 결과를 그대로 게시하지 마세요.**

---

## 1. 안전 보장 부재

- 본 도구의 `riskScore` 0~100은 **패턴 기반 추정**입니다.
- 'low (🟢)' 등급이라도 외부 채널 운영진 재량에 따라 정지·차단될 수 있습니다.
- 본 도구는 외부 채널 정책 변경을 실시간 추적하지 않습니다.

## 2. 외부 채널별 위험 명시

### 🔴 High Tier (검증된 ban 정책)

다음 채널은 자기 홍보를 **명시적으로 금지**합니다. 본 도구는 변환 결과를 제공하지만, 게시 책임은 전적으로 사용자에게 있습니다.

| 채널 | 정책 | 위험 |
|------|------|------|
| 디시인사이드 | "홍보성 타사이트 링크 자동 차단" 공식 정책 | 영구정지 + IP 차단 사례 다수 |
| 더쿠 | 자기 홍보 게시 명시 금지 | 정지 + 활동 제한 |
| 루리웹 | 유료 홍보·반복 게시 금지 | 정지 |
| 셀인형 네이버 카페 | 외부 링크 게시 금지 | 강퇴 + 영구 탈퇴 |

본 도구가 생성하는 "정보 공유체 위장" 패턴은 **운영 노하우(5년+ 활동·고정닉 평판)가 없으면** 거의 100% 들통납니다.

### ⛔ Critical (riskScore ≥ 86)

생성 결과가 critical band로 평가되면 본 도구는 **복사·발행 버튼을 비활성화**합니다. 사용자가 수동 편집 후에만 진행할 수 있습니다.

### ⚫ Out-of-Scope (본 도구 비추천)

| 채널 | 사유 |
|------|------|
| LinkedIn 자동화 | ToS 위반 (수동 게시는 허용) |
| 82쿡 신규 가입 | 가입 폐쇄 10년+ (기존 회원 사용 가능) |
| Reddit 신규 계정 | karma·계정 연차 부족 시 즉시 ban |

## 3. 법적 책임

본 도구의 변환 결과를 게시함으로써 발생한 다음의 모든 책임은 **사용자**가 부담합니다.

- 외부 채널 ToS 위반에 따른 정지·차단·소송
- 한국 법규(표시광고법·부정경쟁방지법 등) 위반에 따른 행정·민사·형사 책임
- 외부 채널 운영진과의 분쟁
- 사용자가 의도하지 않은 후킹·과장 표현으로 인한 명예훼손·허위사실 유포

## 4. 데이터 보안 한계

- 본 도구는 로컬 데이터(consent·로그·피드백)를 `safeStorage` 기반 AES 암호화로 보존합니다.
- OS 키체인 미사용 환경(Linux 일부 배포판 등)에서는 평문 저장될 수 있습니다.
- 사용자 PC가 침해될 경우 로컬 데이터 노출 위험은 사용자 부담입니다.

## 5. 비용 한계

- LLM 호출 비용(Gemini·OpenAI·Claude)은 사용자 API 키 기반이며, 본 도구는 비용 정산 책임이 없습니다.
- 본 도구는 월간 비용 상한 UX를 제공하지만, 실제 청구는 LLM 제공자가 수행합니다.

## 6. 베타 상태 명시

- 본 도구의 외부유입 글 생성 기능은 **베타** 상태입니다.
- 외부 법무 자문이 **미완료**된 상태로 제공됩니다.
- 정식 v1.0 출시 시까지 사용자는 위 모든 위험을 인지하고 자기 책임으로 사용합니다.

## 7. 본 면책 조항의 한계

- 본 면책 조항이 한국 약관규제법상 일방적 면책으로 해석되어 무효가 될 가능성이 있습니다.
- 그 경우에도 본 도구의 위험성 안내·사용자 책임 명시·동의 절차는 사용자가 본 도구의 한계를 인지했음을 입증하는 자료로 활용될 수 있습니다.
